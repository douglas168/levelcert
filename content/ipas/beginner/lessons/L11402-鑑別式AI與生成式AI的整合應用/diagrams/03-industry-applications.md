─────────────────────────────────────────────────────────────────────────────

## 各行業整合流程圖

### 醫療（Healthcare）

```
  CT/X光影像輸入
         ↓
  ┌─────────────────┐
  │ 醫療影像診斷     │  鑑別式AI：偵測腫瘤/病灶位置
  │ （鑑別式）       │
  └────────┬────────┘
           ↓ 偵測結果（腫瘤位置、可疑程度）
  ┌─────────────────┐
  │ 診斷報告生成     │  生成式AI：生成結構化診斷報告草稿
  │ （生成式）       │  ⚠️ 需醫師確認後才發給病患
  └────────┬────────┘
           ↓
  醫師確認後的最終報告
```

### 金融（Finance）

```
  交易資料輸入
         ↓
  ┌─────────────────┐
  │ 詐欺偵測         │  鑑別式AI：判斷是否異常交易
  │ （鑑別式）       │  輸出：fraud_score = 0.87
  └────────┬────────┘
           ↓ 偵測結果
  ┌─────────────────┐
  │ 風險報告生成     │  生成式AI：根據偵測結果撰寫風險說明
  │ （生成式）       │  「本次交易涉及...建議採取...」
  └────────┬────────┘
           ↓
  合規人員審查報告
```

### 教育（Education）

```
  學生答題記錄
         ↓
  ┌─────────────────┐
  │ 弱點分析         │  鑑別式AI：識別學生的薄弱知識點
  │ （鑑別式）       │  輸出：弱點=[統計學, 機率]
  └────────┬────────┘
           ↓ 弱點清單
  ┌─────────────────┐
  │ 個人化題目生成   │  生成式AI：針對弱點生成客製化練習題
  │ （生成式）       │
  └────────┬────────┘
           ↓
  個人化練習題組
```

### 零售（Retail）

```
  商品圖片上傳
         ↓
  ┌─────────────────┐
  │ 商品自動分類     │  鑑別式AI：辨識商品類別/屬性
  │ （鑑別式）       │  輸出：類別=女裝, 顏色=紅, 風格=休閒
  └────────┬────────┘
           ↓ 分類結果
  ┌─────────────────┐
  │ 商品描述生成     │  生成式AI：根據分類結果生成商品文案
  │ （生成式）       │  「這款紅色休閒女裝...」
  └────────┬────────┘
           ↓
  自動完成的商品頁面
```

### 製造（Manufacturing）

```
  產品影像輸入（產線即時）
         ↓
  ┌─────────────────┐        ┌─────────────────────────┐
  │ 瑕疵偵測         │        │  GAN 合成稀有瑕疵圖片    │
  │ （鑑別式）       │ ←── 強化模式：合成資料訓練  ──│  生成式AI              │
  │ 良品/不良品分類  │        └─────────────────────────┘
  └────────┬────────┘
           ↓ 不良品偵測
  ┌─────────────────┐
  │ 維修建議摘要     │  生成式AI：從維修手冊提取並摘要
  │ （生成式）       │
  └────────┬────────┘
           ↓
  維修人員處理指引
```
