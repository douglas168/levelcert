## 場景對照速查

| 場景描述 | 推薦方式 | 判斷依據 |
|---|---|---|
| 大學社團活動報名系統 | No Code | 簡單 + 非技術使用者 |
| 醫院掛號與病歷查詢系統 | Low Code | 安全合規（個資法）|
| 電商促銷活動落地頁 | No Code | 快速上線、頻繁變動 |
| 供應商管理 + 資安稽核 | Low Code | 有合規約束 |
| 銀行核心交易引擎（每秒萬筆） | Traditional | 高效能 + 核心業務 |
| 人事請假申請表單 | No Code | 標準化流程、簡單 |
| 企業 HR 系統（薪資/考核） | Buy（購買軟體） | 標準化功能 + 快速部署 |
| 公司核心 AI 推薦演算法 | Traditional | 核心競爭力 + 複雜演算法 |

## 口訣

> **「安全選 Low，簡單選 No，核心選 Traditional，標準化選 Buy」**
