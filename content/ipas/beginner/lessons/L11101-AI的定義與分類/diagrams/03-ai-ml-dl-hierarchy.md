## 包含關係速查

```
AI ⊃ ML ⊃ DL（大 ⊃ 中 ⊃ 小）

所有 DL 都是 ML   ✅
所有 ML 都是 AI   ✅
所有 DL 都是 AI   ✅

所有 ML 都是 DL   ❌（ML 比 DL 大！ML 還包含決策樹、SVM 等）
所有 AI 都是 ML   ❌（AI 還包含規則式 AI！）
```

## 台灣行政區比喻

```
AI = 台灣（最大範圍）
  ↓
ML = 台北市（台灣的一部分）
  ↓
DL = 信義區（台北市的一部分）

信義區 ⊂ 台北市 ⊂ 台灣
DL     ⊂ ML     ⊂ AI
```

## 考試重點提醒

| 題目問法 | 正確答案 |
|---------|---------|
| AI 和 ML 是同一件事嗎？ | ❌ 不是。ML 是 AI 的子集，AI 還包含規則式 AI |
| 早期的專家系統是 ML 嗎？ | ❌ 不是。專家系統是規則式 AI，屬於 AI 但不是 ML |
| 所有深度學習都是機器學習嗎？ | ✅ 是。DL ⊂ ML |
| ChatGPT 屬於 DL 嗎？ | ✅ 是（使用 Transformer/DL），同時也是 ML 和 AI |
