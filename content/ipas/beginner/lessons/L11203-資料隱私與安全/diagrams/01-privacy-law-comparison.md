# 圖 1：台灣個資法 vs GDPR 主要差異對照表

```
┌────────────────────────┬───────────────────────────────┬───────────────────────────────────────┐
│ 比較面向               │ 台灣個人資料保護法（個資法）    │ GDPR（一般資料保護規則）               │
├────────────────────────┼───────────────────────────────┼───────────────────────────────────────┤
│ 適用地域               │ 台灣境內蒐集/處理/利用          │ 向歐盟居民提供服務即適用（跨境）       │
├────────────────────────┼───────────────────────────────┼───────────────────────────────────────┤
│ 主管機關               │ PDPC（個人資料保護委員會）       │ 各會員國設 DPA                        │
│                        │ 2023年12月籌備處揭牌            │ （Data Protection Authority）         │
├────────────────────────┼───────────────────────────────┼───────────────────────────────────────┤
│ 外洩通報時限           │ 知悉後 72 小時（2025修法，      │ 知悉後 72 小時                        │
│                        │ 時限依子法，與GDPR看齊）        │ 向監管機關通報（Art.33）              │
├────────────────────────┼───────────────────────────────┼───────────────────────────────────────┤
│ 特種/敏感個資類別      │ 6 類                            │ Art.9：8 類                           │
│                        │ 病歷/醫療/基因/性生活/           │ （多含種族/宗教/政治/工會/生物辨識）   │
│                        │ 健康檢查/犯罪前科               │ Art.10：犯罪前科（獨立條文）          │
│                        │ ⚠️ 無種族/宗教/政治            │                                       │
├────────────────────────┼───────────────────────────────┼───────────────────────────────────────┤
│ 當事人法定權利數       │ 5 項（查複補停刪）               │ 超過 5 項                             │
│                        │ ⚠️ 無資料可攜權                │ 多含資料可攜權、被遺忘權、反對權等    │
├────────────────────────┼───────────────────────────────┼───────────────────────────────────────┤
│ 隱私原則基礎           │ 參考 OECD 8 原則                │ Art.5：7 項原則                       │
│                        │ 台灣個資法無明確列出原則數       │ 合法性/目的限制/最小化/正確性/        │
│                        │                                 │ 儲存限制/完整性與機密性/當責性        │
├────────────────────────┼───────────────────────────────┼───────────────────────────────────────┤
│ 最高罰則               │ 非公務機關：2萬～200萬 NTD      │ 最高 2,000 萬歐元                     │
│                        │ （2025修法提升）                │ 或全球年營收 4%（取較高者）           │
└────────────────────────┴───────────────────────────────┴───────────────────────────────────────┘
```

## 考試重點提示

| 考題類型 | 答題方向 |
|---|---|
| 「哪項是 GDPR 有但台灣個資法沒有的特種個資？」 | 種族、宗教信仰、政治見解、工會成員 |
| 「哪項是台灣個資法有但 GDPR 沒有的制度差異？」 | 台灣特種個資含犯罪前科（GDPR 用獨立 Art.10），台灣罰則以 NTD 計 |
| 「台灣個資法當事人有資料可攜權嗎？」 | 否，這是 GDPR 才有的權利 |
| 「兩者外洩通報時限相同嗎？」 | 相同（均為 72 小時），2025 修法後台灣與 GDPR 看齊 |
