# Diagram 02 — Supervised Model Comparison（監督式學習模型比較表）

> Source: L11302 Section 3 · 核心概念 + Section 4 · 易混淆概念對照表

## 主要監督式學習模型比較

| 模型 | 適用任務 | 最適資料規模 | 解釋性 | 計算成本 | 最大特點 |
|---|---|---|---|---|---|
| **線性迴歸** | 迴歸 | 小~大 | ⭐⭐⭐⭐⭐ 最高 | 低 | 預測連續數值；最好理解 |
| **邏輯迴歸** | 分類（尤其二元） | 小~中 | ⭐⭐⭐⭐⭐ 最高 | 低 | 名字有迴歸但任務是分類！ |
| **決策樹** | 分類（也可迴歸） | 小~中 | ⭐⭐⭐⭐⭐ 最高 | 低~中 | 易過擬合；像問答流程圖 |
| **隨機森林** | 分類（也可迴歸） | 中~大 | ⭐⭐ | 中 | Bagging；比決策樹穩定 |
| **SVM** | 分類（也可迴歸） | 小~中 | ⭐⭐ | 中（大資料慢）| 最大邊界；高維特徵強 |
| **KNN** | 分類（也可迴歸） | 小（低維）| ⭐⭐⭐ | 訓練低/預測高 | 懶惰學習；不建模型 |
| **Gradient Boosting / XGBoost** | 分類（也可迴歸）| 中~大 | ⭐ 最低 | 中~高 | Boosting；競賽冠軍模型 |

## 解釋性 vs 準確度 光譜

```
解釋性最高                                         準確度潛力最高
（透明 / 可讀）                                    （複雜 / 黑盒）
    |                                                   |
    ▼                                                   ▼
  線性   邏輯    決策    隨機   SVM   KNN  梯度      深度
  迴歸   迴歸     樹     森林                提升   神經網路
```

> ⚠️ 注意：準確度高低依賴資料特性與調參，上圖僅示意大致趨勢，非絕對排名。

```
低風險 ◄─────────────────────────► 高風險
  邏輯迴歸   線性迴歸   隨機森林   KNN   決策樹（未剪枝）
```

## 考試快速判斷

- 「需要向監管單位解釋決策原因」→ **邏輯迴歸 / 決策樹**
- 「追求最高準確率」→ **隨機森林 / XGBoost**
- 「訓練資料少、不想花時間訓練」→ **KNN（懶惰學習）**
- 「影像分類大量資料」→ **CNN（深度學習）**
