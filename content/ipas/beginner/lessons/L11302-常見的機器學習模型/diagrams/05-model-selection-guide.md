# Diagram 05 — Model Selection Guide（模型選擇決策流程圖）

## ASCII 決策流程圖

```
題目情境
  │
  ╠═══ 資料有標籤嗎？
  │       │
  │       ├─── 有標籤 ──► 監督式學習
  │       │                  │
  │       │                  ╠═══ 輸出是連續數值（金額、溫度）？
  │       │                  │       └─── 迴歸任務
  │       │                  │               ├─ 需要解釋性 ──► 線性迴歸
  │       │                  │               └─ 追求準確度 ──► 隨機森林迴歸 / XGBoost
  │       │                  │
  │       │                  └═══ 輸出是類別（是/否、A/B/C）？
  │       │                          └─── 分類任務
  │       │                                  ├─ 需要高解釋性 ──► 邏輯迴歸 / 決策樹
  │       │                                  ├─ 追求高準確度 ──► 隨機森林 / XGBoost
  │       │                                  ├─ 資料量小（低維）──► KNN / SVM
  │       │                                  └─ 影像/語音/文字大量 ──► CNN / RNN（深度學習）
  │       │
  │       └─── 無標籤 ──► 非監督式學習
  │                          │
  │                          ├─ 想找群組？
  │                          │     ├─ 知道要分幾群 ──► K-means
  │                          │     └─ 不知道分幾群 ──► 階層式分群（Hierarchical Clustering）
  │                          │
  │                          └─ 想壓縮維度 / 視覺化？
  │                                └─ 降維 ──► PCA（主成分分析）
  │
  └═══ 特殊情境
          ├─ 影像辨識（有標籤大量影像）──► CNN
          ├─ 需要可解釋性（金融/醫療監管）──► 邏輯迴歸 / 決策樹
          └─ 模型已訓練好、需要快速預測 ──► SVM（積極學習）比 KNN（懶惰學習）更快
```

## 關鍵詞 → 模型速查卡

```
┌─────────────────────────────────────────────────────────────────┐
│  題目關鍵詞                │  對應模型                           │
├────────────────────────────┼─────────────────────────────────────┤
│  預測房價 / 銷售額（數值） │  線性迴歸                           │
│  垃圾郵件 / 詐騙（是/否）  │  邏輯迴歸                           │
│  流程圖、規則、可解釋分類  │  決策樹                             │
│  集成、Bagging、多棵樹投票 │  隨機森林                           │
│  Boosting、依序修補弱分類  │  Gradient Boosting / XGBoost        │
│  懶惰學習、看鄰居投票      │  KNN                                │
│  最大邊界、超平面          │  SVM                                │
│  無標籤、分群、質心        │  K-means                            │
│  不指定 K、Dendrogram      │  階層式分群                         │
│  降維、解釋方差比          │  PCA                                │
│  影像辨識、卷積            │  CNN                                │
│  序列、時間序列、文字      │  RNN / LSTM                         │
└─────────────────────────────────────────────────────────────────┘
```

## 三大情境練習（快速確認）

```
情境 A：銀行信用評分（10 萬筆，需向監管解釋理由）
  → 有標籤 → 分類 → 需高解釋性 → 邏輯迴歸 / 決策樹 ✓

情境 B：電商顧客分群（50 萬筆，無標籤，找消費族群）
  → 無標籤 → 分群 → 大型資料 → K-means ✓

情境 C：工廠瑕疵品影像辨識（20 萬張，已標記良品/不良品）
  → 有標籤 → 分類 → 影像資料 → CNN（深度學習）✓
```

## 選擇時的三個核心問題

```
1. 有沒有標籤？
   有  → 監督式（分類/迴歸）
   沒有 → 非監督式（分群/降維）

2. 需要解釋性嗎？
   是  → 邏輯迴歸 / 決策樹
   否  → 可考慮隨機森林 / XGBoost / 深度學習

3. 資料型態是什麼？
   影像 → CNN
   序列 → RNN / LSTM
   結構化表格 → 傳統 ML（決策樹、隨機森林、XGBoost）
```
