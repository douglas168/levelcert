# Diagram 03 · Microsoft Copilot 生態系辨別圖

> ⚠️ 考試最高頻混淆點：三個都叫 Copilot，但是完全不同的產品！

```
Microsoft AI 工具生態系（都叫 "Copilot" 但定位不同）
═══════════════════════════════════════════════════════════════════

① Microsoft Copilot
   ┌─────────────────────────────────────────────────────────┐
   │  定位：Office 365 AI 助手                                │
   │  整合：Word / Excel / PowerPoint / Teams / Outlook       │
   │  用戶：一般辦公室工作者                                   │
   │  功能：幫你寫文件、整理會議記錄、分析試算表               │
   │  考試關鍵字：Office、辦公室應用、M365                     │
   └─────────────────────────────────────────────────────────┘

② GitHub Copilot
   ┌─────────────────────────────────────────────────────────┐
   │  定位：IDE 程式碼補全擴充套件（Extension）                │
   │  整合：VS Code / JetBrains / Visual Studio 等 IDE        │
   │  用戶：軟體開發者（程式設計師）                           │
   │  功能：行內程式碼自動補全 + AI Chat + 多檔案上下文        │
   │  價格：Free（有限額）/ Pro $10/月 / Pro+ $39/月           │
   │  考試關鍵字：IDE 擴充套件、行內補全、Extension、程式碼   │
   └─────────────────────────────────────────────────────────┘
           │
           │ 安裝在
           ▼
   ┌─────────────────────────────────────────────────────────┐
   │  VS Code for Copilot                                     │
   │  （非獨立產品，就是 GitHub Copilot 在 VS Code 的介面）    │
   └─────────────────────────────────────────────────────────┘

③ Copilot Studio
   ┌─────────────────────────────────────────────────────────┐
   │  定位：低程式碼 AI Agent 建置平台                         │
   │  整合：Microsoft 365 / Power Platform 生態系             │
   │  用戶：非技術人員 / IT 業務人員                           │
   │  功能：拖拉就能建聊天機器人和自動化流程                   │
   │  相關：Power Automate、Power Apps 同系列                 │

── 獨立生態（非 Microsoft 原生）────────────────────────────────

④ Cursor
   ┌─────────────────────────────────────────────────────────┐
   │  定位：獨立 AI 程式碼編輯器（VS Code fork）               │
   │  用戶：開發者（尤其偏好 Vibe Coding 開發方式）            │
   │  功能：用自然語言描述需求，AI 生成完整程式碼              │

   └─────────────────────────────────────────────────────────┘

═══════════════════════════════════════════════════════════════════
```

## 考試判斷速查

| 情境描述 | 正確答案 |
|---|---|
| 「用 Word 寫報告，想讓 AI 幫忙整理」 | Microsoft Copilot |
| 「在 VS Code 裡寫程式，想要行內補全」 | GitHub Copilot |
| 「Copilot 安裝在 VS Code 中的介面」 | VS Code for Copilot（= GitHub Copilot） |
| 「不懂程式，想建客服聊天機器人」 | Copilot Studio |
| 「用自然語言讓 AI 生成整個登入頁面程式碼」 | Cursor（Vibe Coding） |
| 「Copilot Studio 是 GitHub Copilot 進階版？」 | ❌ 錯！兩者是完全不同產品線 |

## 最重要的一句話

> **「Copilot」這個字在 Microsoft 生態系有三種不同含義。**
> **遇到 Copilot 相關題目，先看「它在哪裡用？用來做什麼？」**
> - 辦公室文件 → Microsoft Copilot
> - 寫程式 → GitHub Copilot
> - 建機器人 → Copilot Studio
