## 快速辨識重點

### 「唯一沒有免費方案」的主流工具
```
Midjourney ← 最低 Basic $10/月，無任何免費方案
```

### 「開源可免費本地部署」的代表工具
```
文字領域：LLaMA（Meta）
圖像領域：Stable Diffusion（Stability AI）
語音辨識：Whisper（OpenAI）← 有開源版本
```

### 「台灣本土」工具
```
TAIDE — 國科會主導，繁體中文 LLM，幾乎每場必考
```

### 計費方式快速對比
```
月費訂閱（固定費）：ChatGPT、Midjourney、GitHub Copilot、Cursor
按量計費（Token）：OpenAI API（用多少付多少）
完全免費：LLaMA（開源）、Stable Diffusion（開源）、TAIDE
```
