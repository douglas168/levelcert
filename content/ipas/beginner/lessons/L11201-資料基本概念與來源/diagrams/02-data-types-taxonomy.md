# 資料型態分類圖（Data Types Taxonomy）

```
資料型態（Data Types）
按「本質」分類
│
├── 📊 數值型（Numerical Data）
│   ├── 離散型（Discrete）— 可數，只能是整數
│   │   └── 例：7-11 今天賣 87 杯咖啡、IG 粉絲 1,203 人
│   └── 連續型（Continuous）— 可測量，可以有小數
│       └── 例：氣溫 28.5°C、體重 65.3 kg
│   → 對應 AI 技術：迴歸分析（Regression）、預測（Prediction）
│
├── 📝 文字型（Text Data）
│   └── 人類語言產生的文字內容，無固定格式
│       └── 例：LINE 訊息、PTT 文章、蝦皮評價「很好穿！推推」
│   → 對應 AI 技術：自然語言處理（NLP, Natural Language Processing）
│
├── 🖼️ 圖像型（Image Data）
│   └── 像素（Pixel）矩陣組成的靜態視覺資料
│       └── 例：Instagram 照片、工廠產品照、X 光片
│   → 對應 AI 技術：電腦視覺（Computer Vision, CV）
│
├── 🔊 音訊型（Audio Data）
│   └── 聲音信號，含頻率、音高、音量等特徵
│       └── 例：Siri 語音輸入、客服電話錄音、Spotify 音樂
│   → 對應 AI 技術：語音辨識（Speech Recognition）
│
├── 🎬 影片型（Video Data）
│   └── 連續圖像序列（Image Sequence）+ 音訊的組合
│       └── 例：YouTube 影片、路口監視器串流、健身 App 動作影片
│   → 對應 AI 技術：電腦視覺（CV）+ 語音辨識（Audio Processing）
│
└── 🏷️ 類別型（Categorical Data）
    └── 表示分組或分類，不能做數學運算
        └── 例：血型（A/B/O/AB）、商品類別（3C/服飾）、VIP/一般會員
    → 對應 AI 技術：分類（Classification）、推薦系統（Recommendation）
```

## 型態 × AI 技術 × 台灣應用 速查表

| 資料型態 | 對應 AI 技術 | 台灣生活舉例 | 蝦皮/104/7-11 情境 |
|---------|------------|------------|-------------------|
| 數值型 | 迴歸/預測 | 全聯預測補貨量 | 蝦皮訂單金額（數值）→ 銷售預測 |
| 文字型 | NLP（自然語言處理） | 蝦皮評論情感分析 | 104 履歷文字 → AI 職缺配對 |
| 圖像型 | 電腦視覺（CV） | 台積電晶圓瑕疵檢測 | 7-11 貨架照片 → 自動盤點 |
| 音訊型 | 語音辨識 | 中華電信客服轉文字 | LINE 語音訊息 → 文字回覆 |
| 影片型 | CV + 音訊處理 | 捷運站動作辨識 | YouTube 影片 → 內容審核 |
| 類別型 | 分類/推薦 | LINE 購物個人化推薦 | 蝦皮商品分類標籤 → 推薦 |

## 離散型 vs 連續型 快速判斷

```
看到「計數」類數字 → 離散型（Discrete）
  ✓ 今天賣了 87 杯咖啡
  ✓ 班上有 32 個同學
  ✓ IG 粉絲數 1,203 人
  ✗ 不會有「2.5 杯咖啡」

看到「測量」類數字 → 連續型（Continuous）
  ✓ 氣溫 28.5°C（可以更精確到 28.52°C）
  ✓ 月薪 45,230 元（任意金額）
  ✓ 身高 168.3 cm
  ✓ 可以無限精確
```
