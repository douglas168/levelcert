# 圖解 03 — Transformer 系列定位圖（GPT vs BERT）

> 初級考點：GPT（Decoder-only，單向，生成式）vs BERT（Encoder-only，雙向，鑑別式任務）。不考 self-attention 公式。

## LLM 家族定位

## 文字方向性示意

```
GPT（單向 → 自迴歸）：
    我 → 喜 → 歡 → 吃 → [下一個token?]
    只能看左邊已出現的字，逐字生成

BERT（雙向 ← → ）：
    我 [MASK] 歡 吃 蘋果
    ← 同時看左右 →
    填空：「喜」（從兩側上下文推斷）
```

## 快速對比表

| 項目 | GPT | BERT |
|---|---|---|
| 架構 | Decoder-only | Encoder-only |
| 方向 | 單向（左→右） | 雙向（←→） |
| 訓練任務 | 預測下一個 token | 填空（Masked LM） |
| 主要用途 | 文字生成 / 對話 | 分類 / 問答 / 理解 |
| AI 類型 | 生成式 | 主要做鑑別式任務 |
| 代表應用 | ChatGPT, GPT-4o | 搜尋引擎、情感分析 |

## ⚠️ 考試陷阱

```
❌ 陷阱 1：「BERT 是語言模型，所以是生成式AI」
   → BERT 的用途是理解和分類，不是生成文字

❌ 陷阱 2：「GPT 是雙向模型」
   → GPT 是單向（Unidirectional）；雙向是 BERT

❌ 陷阱 3：「Decoder-only = 鑑別式」
   → Decoder-only（如 GPT）= 生成式；Encoder-only（如 BERT）= 鑑別式任務

記憶口訣：「GPT 單向生，BERT 雙向懂」
```
