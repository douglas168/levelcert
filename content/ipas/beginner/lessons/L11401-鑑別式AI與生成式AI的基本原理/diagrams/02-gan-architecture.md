# 圖解 02 — GAN 生成對抗網路概念圖

> 初級考點：GAN 屬於生成式AI，有兩個角色（生成器 + 鑑別器）相互對抗。不考訓練損失函數的數學細節。

```
                        GAN（Generative Adversarial Network，生成對抗網路）
                        ───────────────────────────────────────────────

 隨機雜訊（Noise）
      │
      ▼
 ┌──────────────────┐         生成假圖         ┌──────────────────────┐
 │                  │  ─────────────────────→  │                      │
 │  生成器           │                          │  鑑別器               │
 │  Generator       │                          │  Discriminator       │
 │                  │                          │                      │
 │  目標：          │                          │  目標：              │
 │  生成「以假亂真」 │                          │  判斷輸入是真實還是  │
 │  的圖像          │                          │  生成器造假的        │
 └──────────────────┘                          └──────────────────────┘
                                                        │
                                                        │ 真實資料
                                                        ▼
                                               ┌────────────────┐
                                               │  真實訓練資料   │
                                               │  Real Data     │
                                               └────────────────┘

 ← ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ 對抗反饋（讓生成器越來越好）─ ─ ─ ─ ─ ─ ─ ─ ─ ─→
```

## 兩個角色的比喻

```
┌────────────────────────────────────────────────────────────┐
│  警察與罪犯比喻                                              │
│                                                            │
│  生成器 = 偽鈔製造商                                        │
│    → 目標：製作出讓銀行驗不出來的假鈔                        │
│                                                            │
│  鑑別器 = 銀行驗鈔機                                        │
│    → 目標：辨別出哪些是真鈔、哪些是假鈔                      │
│                                                            │
│  兩者互相競爭，最終生成器學會製造以假亂真的圖像              │
└────────────────────────────────────────────────────────────┘
```

## GAN 典型應用

| 應用 | 說明 |
|---|---|
| DeepFake 圖像生成 | 生成逼真的人臉圖像 |
| 資料擴增（Data Augmentation） | 為訓練集生成額外的合成資料 |
| 圖像風格轉換 | 將照片轉換成畫風 |
| 醫療影像合成 | 生成稀缺的醫療訓練資料 |

## ⚠️ 考試注意

```
GAN 整體架構 = 生成式AI ✓（Generative 是名字第一個字）

但 GAN 裡面的 Discriminator 是做「分類/判斷」（鑑別）
    → 它是生成式整體架構中的一個「輔助判斷」組件
    → 不能因為有 Discriminator 就說 GAN 是鑑別式

只有 GAN 有 Generator + Discriminator 對抗結構。
VAE 和 Diffusion Model 沒有這種對抗架構。
```
