# 圖解 01 — 鑑別式AI vs 生成式AI 核心對比

```
┌─────────────────────────────────────────────────────────────────────┐
│                  AI 模型兩大方向                                      │
├───────────────────────────┬─────────────────────────────────────────┤
│   鑑別式AI                │   生成式AI                               │
│   Discriminative AI       │   Generative AI                         │
├───────────────────────────┼─────────────────────────────────────────┤
│ 核心問題                  │ 核心問題                                  │
│  「這是什麼？」            │  「幫我創造！」                           │
│  「屬於哪一類？」          │  「生成一個新的！」                       │
├───────────────────────────┼─────────────────────────────────────────┤
│ 學習目標                  │ 學習目標                                  │
│  找決策邊界               │  學資料分布                               │
│  Decision Boundary        │  Data Distribution                      │
├───────────────────────────┼─────────────────────────────────────────┤
│ 資料需求                  │ 資料需求                                  │
│  需要標註資料（Labeled）   │  可用未標註資料（Unlabeled）              │
├───────────────────────────┼─────────────────────────────────────────┤
│ 輸出                      │ 輸出                                     │
│  類別 / 數值              │  全新文字 / 圖像 / 音頻 / 影片 / 程式碼  │
├───────────────────────────┼─────────────────────────────────────────┤
│ 代表模型                  │ 代表模型                                  │
│  SVM                      │  GAN（Generative Adversarial Network）  │
│  決策樹（Decision Tree）   │  VAE（Variational Autoencoder）⚠️       │
│  邏輯迴歸（Logistic Reg.）│  Diffusion Model（擴散模型）             │
│  隨機森林（Random Forest） │  GPT（Generative Pre-trained Transf.）  │
├───────────────────────────┼─────────────────────────────────────────┤
│ 典型應用                  │ 典型應用                                  │
│  垃圾郵件過濾              │  ChatGPT（文字生成）                     │
│  詐騙交易偵測              │  DALL-E / Midjourney（圖像生成）         │
│  醫療影像分類              │  GitHub Copilot（程式碼生成）            │
│  情感分析                  │  Sora（影片生成）                        │
├───────────────────────────┼─────────────────────────────────────────┤
│ 主要風險                  │ 主要風險                                  │
│  分類錯誤、偏見            │  幻覺（Hallucination）、偏見放大         │
└───────────────────────────┴─────────────────────────────────────────┘
```

## ⚠️ 易混淆陷阱

```
VAE 名字有 "Autoencoder"  →  聽起來像在「分析」  →  其實是生成式 ✓
BERT 名字有 "Language Model" →  聽起來像生成  →  主要做鑑別式任務 ✓
GAN 有 Discriminator 角色  →  聽起來是鑑別  →  整體架構是生成式 ✓
```

## 快速判斷流程

```
題目情境
    │
    ├── 任務是「判斷/分類/預測」？ ──→ 鑑別式AI
    │
    └── 任務是「生成/創造/寫/畫」？ ──→ 生成式AI
```
