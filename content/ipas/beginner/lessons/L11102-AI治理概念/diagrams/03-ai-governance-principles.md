## 框架層次關係

```
   【非強制性框架（倫理/原則層）】
   ┌─────────────────────────────────────┐
   │  OECD AI Principles（2019/2024）    │  ← 各國立法的共同基礎
   │  UNESCO AI Ethics（2021，193國）    │  ← 全球最廣泛倫理框架
   └─────────────────────────────────────┘
                      ↓ 各國將原則轉化為法規
   【具法律約束力框架（監管/法律層）】
   ┌─────────────────────────────────────┐
   │  EU AI Act（2024）                  │  ← 全球首部AI法律
   │  台灣《人工智慧基本法》（2026）       │  ← 台灣首部AI專法
   │  中國《生成式AI管理辦法》（2023）     │  ← 全球首個生成式AI法規
   └─────────────────────────────────────┘
                      ↓ 各部會出分領域指引
   【行政指導（產業層）】
   ┌─────────────────────────────────────┐
   │  金管會《金融業AI指引》（非強制）     │  ← 金融業
   │  數位發展部《公部門手冊》            │  ← 政府機關
   └─────────────────────────────────────┘
```

## 🔑 考試重點

- OECD + UNESCO = **非強制性**（原則/倫理框架）
- EU AI Act = **最嚴格的具法律約束力框架**
- 台灣基本法 = **7條**，金管會指引 = **6條**，OECD = **5條**
- 台灣基本法多出的一條原則（相較金管會指引）= **③隱私保護與資料治理**
