# 生成式AI 風險分類樹 — Risk Taxonomy Tree

> L12303 生成式AI風險管理 | 初級 Planner 視角
> 三大主類：倫理風險 / 資安隱私 / 合規性

```
生成式AI 風險（GenAI Risks）
│
├── 🎭 倫理風險（Ethical Risks）                           🔥🔥🔥
│    │
│    ├── 幻覺（Hallucination）
│    │     定義：模型生成不存在的事實，語氣肯定、外觀合理
│    │     根因：機率式 token 生成本質（不是故意撒謊）
│    │     緩解：RAG + 來源引用 + HITL 覆核
│    │     考點：幻覺 ≠ 謊言 ≠ 偏見（根因不同）
│    │
│    ├── 偏見（Bias / Discrimination）
│    │     定義：訓練資料不均衡→模型複製放大不公平傾向
│    │     根因：訓練資料（不是「模型壞掉」）
│    │     緩解：資料多樣性審查 + 公平性測試
│    │     考點：iPAS 樣題明確：偏見/歧視 = 倫理風險 🔥
│    │
│    ├── 深偽（Deepfake）                                  🔥🔥
│    │     定義：用 GenAI 偽造真人臉/聲音/影片
│    │     特性：同時是倫理 + 資安 + 法律三類風險
│    │     台灣案例：TSMC 假高管、假縣長農產品代言
│    │
│    ├── 著作權侵害（Copyright Infringement）              🔥
│    │     台灣智慧局 2023-06 函釋：
│    │     純 AI 產出 → 不受著作權保護
│    │     訓練抓取他人著作 → 屬「重製」須授權
│    │
│    ├── 濫用（Misuse）
│    │     定義：合法工具用於非法/惡意目的
│    │     例：釣魚信大量生成、詐騙腳本、惡意程式草稿
│    │
│    └── 錯假訊息（Misinformation / Disinformation）
│          定義：人為（可能借助 AI 工具）散布錯誤資訊
│
├── 🔐 資安隱私（Security & Privacy Risks）               🔥🔥🔥
│    │
│    ├── 提示注入（Prompt Injection）OWASP LLM01 第一名
│    │     │
│    │     ├── 直接注入（Direct Injection）
│    │     │     使用者親自打「忘記規則⋯⋯」
│    │     │
│    │     ├── 間接注入（Indirect Injection）             🔥🔥
│    │     │     惡意指令藏在 PDF/網頁/郵件/RAG 資料源
│    │     │
│    │     └── 越獄（Jailbreak）
│    │           用角色扮演繞過安全限制
│    │
│    ├── 提示外洩（Prompt Leakage）                       🔥🔥
│    │     定義：員工把機敏資料貼入公有雲 LLM
│    │     典型：三星案（2023）原始碼+會議錄音
│    │
│    ├── 訓練資料外洩（Training Data Leakage）
│    │     定義：模型被觸發後近乎逐字輸出訓練資料（含 PII）
│    │
│    └── 個資曝露（PII Exposure）
│          去識別化 ≠ 零風險（仍可被重新識別）
│
└── ⚖️ 合規性（Compliance Risks）                        🔥🔥
     │
     ├── 人工智慧基本法（AI Basic Act）2025-12-24 三讀  🔥🔥
     │     中央主管：國科會
     │     §16 風險分級框架：由 moda 數位發展部 訂定
     │     七大原則：永續/自主/隱私/資安/透明/公平/問責
     │
     ├── 個資法（Personal Data Protection Act）× GenAI
     │     §20 特定目的外利用
     │     §27 安全維護義務
     │
     └── 著作權法（Copyright Act）× AI 生成物
           智慧局 2023-06 函釋：純 AI 產出不保護
           訓練資料重製：須授權或合理使用依據
```

## 考試快速記憶

| 風險類別 | 代表子項 | 關鍵詞 |
|---|---|---|
| 倫理風險 | 偏見/幻覺/深偽 | 「對人不公平/誤導/身份冒用」|
| 資安隱私 | Prompt Injection/外洩 | 「繞過規則/資料脫離控制」|
| 合規性 | 三法對齊 | 「法規/著作權/個資」|

> 口訣：**幻偏誤權偽**（倫理五類）、**漏注洩**（資安三情境）、**基個著**（三法）
