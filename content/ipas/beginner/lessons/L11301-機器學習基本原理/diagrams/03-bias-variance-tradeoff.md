# 圖示 03：Overfitting vs Underfitting vs Good Fit

> 對應 Study Guide Section 3.5

## 測試集錯誤率 vs 模型複雜度（觀念示意圖）

```
  測試集錯誤率（高）
  │
  │ *                                                *
  │  *                                             *
  │   *                                          *
  │    *                                       *
  │     *                                    *
  │      *                                *
  │       *  擬合不足區             過度擬合區
  │        *                       *
  │         *                   *
  │          *               *
  │           *           *
  │            *       *
  │             *  ★ *
  │               *
  │          最佳區間
  │          （Good Fit）
  │
  (低) ────────────────────────────────────────────▶ 模型複雜度
       (太簡單)        (適中)              (太複雜)
       擬合不足        Good Fit            過度擬合
```

**注意：這條曲線描述的是「測試集」錯誤率，不是訓練集。**

## 三種狀態對照表

```
┌────────────────┬──────────────────────────────┬──────────────────────────────────┐
│                │         訓練集表現              │          測試集表現               │
├────────────────┼──────────────────────────────┼──────────────────────────────────┤
│  擬合不足       │  ❌ 差（還沒學會）              │  ❌ 差（和訓練集差不多差）          │
│  Underfitting  │  例：訓練集 58%                │  例：測試集 56%                   │
├────────────────┼──────────────────────────────┼──────────────────────────────────┤
│  Good Fit      │  ✅ 好                         │  ✅ 好（接近訓練集）               │
│                │  例：訓練集 92%                │  例：測試集 90%                   │
├────────────────┼──────────────────────────────┼──────────────────────────────────┤
│  過度擬合       │  ✅✅ 非常好（近乎完美）          │  ❌ 差（落差大）                  │
│  Overfitting   │  例：訓練集 98%                │  例：測試集 60%                   │
└────────────────┴──────────────────────────────┴──────────────────────────────────┘
```

## 學生比喻

```
  擬合不足 = 根本沒讀書的學生
  ┌─────────────────────────────────────────┐
  │ 平時練習題：55 分                         │
  │ 正式大考：53 分                           │
  │ 兩邊都爛，因為根本沒學                     │
  └─────────────────────────────────────────┘

  Good Fit = 真正理解概念的學生
  ┌─────────────────────────────────────────┐
  │ 平時練習題：92 分                         │
  │ 正式大考：90 分                           │
  │ 兩邊都好，因為學到了真正的規律              │
  └─────────────────────────────────────────┘

  過度擬合 = 死背考古題的學生
  ┌─────────────────────────────────────────┐
  │ 平時練習題：99 分（把答案背下來了）          │
  │ 正式大考：60 分（題目一換就不會了）          │
  │ 高低落差大，因為只是「背答案」              │
  └─────────────────────────────────────────┘
```

## 解法方向（僅認名詞，不需細節）

| 問題 | 解法方向（認名即可） |
|---|---|
| 擬合不足 | 增加模型複雜度、增加特徵、延長訓練時間 |
| 過度擬合 | 收集更多資料、降低複雜度、提前停止（Early Stopping）、正則化（Regularization） |

## 🔥 考試速查

```
  訓練高、測試低（高低差大）→ 過度擬合（Overfitting）
  訓練低、測試低（兩邊都差）→ 擬合不足（Underfitting）
  訓練高、測試也高（接近）  → Good Fit（泛化良好）
```
