## 1. Big Picture / Core Pipeline 🔥🔥🔥

### 先懂一句話

`機率分佈（Probability Distribution）` 是在回答：「某個隨機數字可能出現哪些值？每個值或區間有多可能？」考試通常不是要你先背公式，而是要你從題目文字判斷該用哪一種機率語言。

### Everyday Analogy

把本課想成整理一間便利商店的資料：

- 「每小時來幾位客人」是可以數的件數。
- 「一位客人等待幾分鐘」是連續時間。
- 「剛好 3 位客人」和「5 分鐘內」問的是不同機率。

你先看資料是什麼型態，再選讀機率的方法。

### 先問自己一個問題

題目到底是在問哪一種事情？

```text
1. 這個數字是可數的件數，還是連續量測值？
2. 題目問剛好等於、某段區間，還是累積到某個值？
3. 題目線索比較像固定次數、固定時間事件、鐘形資料，還是區間等可能？
```

### 技術說法

本課核心名詞可以先這樣放進腦中：

| 中文 / English | 先用白話記 |
| -------------- | ---------- |
| 隨機變數（Random Variable, RV） | 把隨機結果變成數字 |
| 離散型（Discrete） | 可以一個一個數 |
| 連續型（Continuous） | 落在連續區間內 |
| PMF（Probability Mass Function） | 離散型的單點機率 |
| PDF（Probability Density Function） | 連續型的密度曲線 |
| CDF（Cumulative Distribution Function） | 累積到某值為止的機率 |

### 流程 / 選擇流程

```text
題目情境
→ 判斷隨機變數型態
→ 選 PMF / PDF / CDF
→ 選 Normal / Binomial / Poisson / Uniform
→ 讀參數或 scipy.stats 程式碼
```

### 比較表這樣讀

這張表不要死背，請把它當作「看到題目關鍵字時往哪裡走」：

| 題目問的是                             | 想到                |
| -------------------------------------- | ------------------- |
| 次數、件數、成功幾次                   | 離散型隨機變數      |
| 時間、長度、重量、溫度                 | 連續型隨機變數      |
| `P(X = x)` 且 X 是離散型               | PMF                 |
| 連續型某區間的機率                     | PDF 面積或 CDF 相減 |
| `P(X ≤ x)`                             | CDF                 |
| 對稱鐘形、`μ`、`σ²`、z-score           | Normal              |
| 固定 `n` 次、成功 / 失敗、成功機率 `p` | Binomial            |
| 固定時間或空間內的事件次數、平均率 `λ` | Poisson             |
| 區間 `[a,b]` 內等可能                  | Uniform             |

### 記憶方式

```text
件數 → 離散 → PMF / Binomial / Poisson
時間與量測 → 連續 → PDF / CDF / Normal / Uniform
剛好等於 → PMF
小於等於 → CDF
區間內 → CDF 相減或 PDF 面積
```

### Exam Rule

```text
題目先問「資料型態」→ 先分離散 / 連續
題目問「剛好等於」→ 離散用 PMF，連續單點機率為 0
題目問「小於等於」→ CDF
題目問「區間內」→ CDF 相減或 PDF 面積
```

### Quick Check

**Q.** 題目說「某客服中心每小時收到幾通客訴」，第一步要把它看成離散型還是連續型？

答案：離散型。因為客訴通數是可以數的件數。

## 2. 隨機變數與離散 / 連續判斷 🔥🔥

### 先懂一句話

`隨機變數（Random Variable, RV）` 是「把隨機結果轉成數字」的規則。第一個考點通常是：這個數字是「可數的」還是「量出來的」。

### Everyday Analogy

像班上抽獎：

- 抽到哪個獎品是隨機結果。
- 如果你記錄「今天有幾個人中獎」，這個數字就是隨機變數。
- 如果你記錄「每個人等待抽獎的時間」，這也是隨機變數，但它是連續型。

### 先問自己一個問題

這個值能不能用 `0, 1, 2, 3...` 這種方式一個一個數？

- 可以數清楚：多半是離散型（Discrete）。
- 是量測出來、可能有小數：多半是連續型（Continuous）。

### 技術說法

`樣本空間（Sample Space）` 是所有可能結果；`事件（Event）` 是你關心的一組結果；`隨機變數（Random Variable）` 是把結果轉成數字的規則。

### 流程 / 選擇流程

```text
樣本空間 / 事件
→ 隨機變數
→ 離散型 or 連續型
→ 選 PMF / PDF / CDF
```

### 比較表這樣讀

這張表先看「白話」和「例子」兩欄。考試通常用例子包裝，不會直接問你定義。

| 概念                        | 白話             | 例子                   | 常接的工具   |
| --------------------------- | ---------------- | ---------------------- | ------------ |
| 樣本空間（Sample Space）    | 所有可能結果     | 抽獎可能抽到的所有獎品 | 定義問題範圍 |
| 事件（Event）               | 你關心的結果集合 | 抽到耳機               | 算機率       |
| 隨機變數（Random Variable） | 把結果映射成數字 | 今天中獎人數           | 接分佈       |
| 離散型（Discrete）          | 取值可數         | 人數、件數、答對題數   | PMF、CDF     |
| 連續型（Continuous）        | 取值落在連續區間 | 時間、身高、重量       | PDF、CDF     |

### 一步一步例子

例子 1：問卷答對題數

```text
X = 一份問卷答對題數
可能值 = 0, 1, 2, 3, ..., 20
可以一個一個數
所以 X 是離散型（Discrete）
```

例子 2：外送送達時間

```text
Y = 外送送達時間
可能值 = 25 分鐘、25.3 分鐘、25.31 分鐘...
可以有小數，落在連續區間
所以 Y 是連續型（Continuous）
```

### 記憶方式

```text
數出來的 → 離散型
量出來的 → 連續型
```

### Exam Rule

```text
次數 / 件數 / 人數 / 成功幾次 → 離散型
時間 / 長度 / 重量 / 溫度 / 分數量測值 → 通常連續型
離散型單點 P(X=x) → 可能大於 0
連續型單點 P(X=x) → 等於 0
```

### Quick Check

**Q.** 「通勤所花分鐘數」在本課通常看成哪一型隨機變數？

答案：連續型。因為時間理論上可以有小數，例如 25.3 分鐘。

## 3. PMF / PDF / CDF 🔥🔥🔥

### 先懂一句話

PMF / PDF / CDF 是讀機率分佈的三種語言：`PMF` 讀離散單點，`PDF` 讀連續密度，`CDF` 讀累積到某個值為止。

### Everyday Analogy

像看店裡排隊：

- PMF：剛好 3 個人排隊的機率。
- PDF：等待時間曲線在某個位置的密度高低。
- CDF：等 5 分鐘以內的人累積有多少。

### 先問自己一個問題

題目問的是哪一種句型？

```text
剛好等於 X=x？
小於等於 X≤x？
介於 a 和 b 之間？
```

### 技術說法

- `PMF（Probability Mass Function，機率質量函數）`：離散型的單點機率，寫成 `P(X=x)`。
- `PDF（Probability Density Function，機率密度函數）`：連續型的密度函數，單點高度不是機率。
- `CDF（Cumulative Distribution Function，累積分布函數）`：累積到 `x` 為止，寫成 `F(x)=P(X≤x)`。

### 流程 / 選擇流程

```text
判斷離散 / 連續
→ 判斷題目問單點 / 區間 / 累積
→ PMF / PDF / CDF
```

### 比較表這樣讀

先看「適用」與「讀法」兩列。這兩列會幫你避免把 PDF 當成單點機率。

| 項目     | PMF                        | PDF                          | CDF                                    |
| -------- | -------------------------- | ---------------------------- | -------------------------------------- |
| 中文     | 機率質量函數               | 機率密度函數                 | 累積分配函數                           |
| 英文     | Probability Mass Function  | Probability Density Function | Cumulative Distribution Function       |
| 適用     | 離散型                     | 連續型                       | 離散型與連續型都可                     |
| 讀法     | `P(X=x)`                   | `f(x)`，密度不是單點機率     | `F(x)=P(X≤x)`                          |
| 機率來源 | 單點機率                   | 區間下面面積                 | 累積機率                               |
| 常見函式 | `binom.pmf`, `poisson.pmf` | `norm.pdf`, `uniform.pdf`    | `norm.cdf`, `binom.cdf`, `poisson.cdf` |

### 一步一步例子

例子 1：PMF 讀離散單點

```text
X = 一小時內收到的客訴件數
題目問 P(X = 3)
X 是件數 → 離散型
問剛好 3 件 → 用 PMF
```

例子 2：CDF 讀累積

```text
F(3) = 0.7
CDF 定義: F(x) = P(X ≤ x)
所以 F(3) = P(X ≤ 3) = 0.7
意思：X 小於等於 3 的機率是 0.7
```

例子 3：連續區間用 CDF 相減

```text
已知 F(8) = 0.90，F(5) = 0.40
題目問 P(5 < X ≤ 8)
P(5 < X ≤ 8) = F(8) - F(5)
                 = 0.90 - 0.40
                 = 0.50
```

### 記憶方式

```text
PMF = Point for discrete = 離散單點
PDF = Density for continuous = 連續密度
CDF = Cumulative = 累積到 x
```

### Exam Rule

```text
看到 P(X = x) + 離散型 → PMF
看到連續型某點 f(x) → PDF 密度，不是機率
看到 P(X ≤ x) / 最多 / 不超過 / 以下 → CDF
看到 P(a < X ≤ b) → F(b) - F(a)
PDF 數值可以大於 1 → 因為它是密度，不是機率
```

### Quick Check

**Q.** 對連續型隨機變數，`P(X = 10)` 等於多少？

答案：0。連續型的機率來自區間面積，不是單一精準點。

## 4. 常態分佈 Normal Distribution 🔥🔥

### 先懂一句話

`常態分佈（Normal Distribution）` 是左右對稱的鐘形連續分佈。題目看到平均附近最多、兩端較少、`μ`、`σ²`、z-score，就先想到 Normal。

### Everyday Analogy

像大型考試成績：大多數人集中在平均分附近，特別高分和特別低分的人比較少，左右大致對稱。

### 先問自己一個問題

題目描述的是不是「連續數值 + 鐘形對稱」？

如果是，才往 Normal 走；如果題目是件數、成功次數或固定時間內事件次數，就不要硬套 Normal。

### 技術說法

記號：

```text
X ~ Normal(μ, σ²)
```

| 符號 | 意思 |
| ---- | ---- |
| `μ` | 平均數（Mean） |
| `σ²` | 變異數（Variance） |
| `σ` | 標準差（Standard Deviation） |

### 流程 / 選擇流程

```text
連續型資料
→ 對稱鐘形 / 平均與標準差
→ Normal(μ, σ²)
→ PDF / CDF / z-score
```

### 一步一步例子

例子：`X ~ Normal(70, 100)`，問 85 分的 z-score。

```text
數學記號: Normal(70, 100)
μ = 70
σ² = 100
σ = sqrt(100) = 10

z = (x - μ) / σ
  = (85 - 70) / 10
  = 15 / 10
  = 1.5
```

意思：85 分比平均數高 1.5 個標準差。

### 比較表這樣讀

這張表是看圖形與參數，不是背長公式：

| 性質       | 考試記法                   |
| ---------- | -------------------------- |
| 對稱於 `μ` | 平均數、中位數、眾數在中心 |
| `σ` 越大   | 曲線越扁越寬               |
| `σ` 越小   | 曲線越高越窄               |
| 可標準化   | `z = (x - μ) / σ`          |

### 記憶方式

68-95-99.7 法則：

```text
μ ± 1σ 內：約 68%
μ ± 2σ 內：約 95%
μ ± 3σ 內：約 99.7%
```

SciPy 讀法：

```python
norm.pdf(x, loc=μ, scale=σ)
norm.cdf(x, loc=μ, scale=σ)
```

注意：`scale` 傳的是 `σ`，不是 `σ²`。

```text
Normal(μ, σ²) → 數學常寫變異數
scipy scale → 要填標準差 σ
```

### Exam Rule

```text
對稱鐘形 / 中間最多兩端少 → Normal
題目給 μ 和 σ² → Normal
題目提 z-score / 標準化 → Normal
題目提 68-95-99.7 → Normal
scipy norm 的 scale → 標準差 σ，不是變異數 σ²
```

### Quick Check

**Q.** `X ~ Normal(70, 100)`，在 `norm.cdf(x, loc=70, scale=?)` 中 `scale` 應填多少？

答案：10。因為數學記號中的 100 是變異數 `σ²`，SciPy 的 `scale` 要填標準差 `σ`。

## 5. 二項分佈 Binomial Distribution 🔥🔥

### 先懂一句話

`二項分佈（Binomial Distribution）` 是在問：「固定做 `n` 次，每次只有成功 / 失敗，最後成功幾次？」

### Everyday Analogy

像寄 10 封履歷：

- 每封結果先簡化成「有面試」或「沒面試」。
- 每封被邀請面試的機率先假設固定。
- 問 10 封裡剛好 3 封有面試，就是 Binomial。

### 先問自己一個問題

題目有沒有同時出現這四件事？

```text
固定 n 次
每次只有成功 / 失敗
成功機率 p 固定
每次結果彼此獨立
```

四個都有，才是典型二項分佈。

### 技術說法

記號：

```text
X ~ Binomial(n, p)
```

| 符號 | 意思 |
| ---- | ---- |
| `n` | 固定試驗次數 |
| `p` | 每次成功機率 |
| `k` | 成功次數 |

### 流程 / 選擇流程

```text
離散型次數
→ 固定 n 次試驗
→ 每次成功 / 失敗
→ Binomial(n, p)
```

### 一步一步例子

例子：投 3 次公平硬幣，問剛好 2 次正面的機率。

```text
n = 3
p = 0.5
k = 2

公式:
P(X = k) = C(n,k) p^k (1-p)^(n-k)

C(3,2) = 3! / [2!(3-2)!]
       = 6 / 2
       = 3

P(X = 2) = C(3,2)(0.5)^2(1-0.5)^(3-2)
         = 3 × 0.25 × 0.5
         = 0.375
```

`C(n,k)` 不能省略，因為它在數「哪幾次成功」有幾種排列。

### 比較表這樣讀

這張表用來檢查題目是不是符合 Binomial 條件：

| 條件 | 問自己 | 例子 |
| ---- | ------ | ---- |
| 固定 `n` 次 | 次數有沒有先固定？ | 打 20 通電話 |
| 兩種結果 | 每次能不能切成成功 / 失敗？ | 成交 / 未成交 |
| 成功機率固定 | 每次機率是否可視為同一個 `p`？ | 每通成交率 10% |
| 各次獨立 | 這次結果是否不影響下次？ | 每通電話近似獨立 |

### 記憶方式

```text
Binomial = 固定 n 次 + 成功/失敗 + 問成功 k 次
```

PMF、平均與變異：

```text
P(X = k) = C(n,k) p^k (1-p)^(n-k)
C(n,k) = n! / [k!(n-k)!]

Mean = np
Variance = np(1-p)
```

`scipy.stats` 讀法：

```python
binom.pmf(k, n, p)  # P(X = k)
binom.cdf(k, n, p)  # P(X ≤ k)
```

### Exam Rule

```text
固定做 n 次 → Binomial
每次只有成功 / 失敗 → Binomial
問成功幾次 → Binomial
完整 PMF 不能漏 C(n,k)
binom.pmf(k,n,p) → 剛好 k 次
binom.cdf(k,n,p) → 最多 k 次
```

### Quick Check

**Q.** 12 題是非題中，問剛好答對 8 題的機率，最像哪個分佈？

答案：二項分佈。因為有固定 12 次試驗，每題可視為答對 / 答錯。

## 6. 卜瓦松分佈與均勻分佈 🔥🔥

### 先懂一句話

`卜瓦松分佈（Poisson Distribution）` 問固定時間 / 空間內事件發生幾次；`均勻分佈（Uniform Distribution）` 問已知區間內每個位置都一樣可能。

### Everyday Analogy

Poisson 像記錄「每小時客服電話幾通」；Uniform 像系統說「驗證碼會在 2 到 6 分鐘內任一時間等可能送達」。

### 先問自己一個問題

題目是在問「固定區間內發生幾次」，還是「區間內每個位置等可能」？

- 發生幾次 + 平均率 `λ` → Poisson。
- `[a,b]` 內每個位置等可能 → Uniform。
- 等下一次事件多久 → Exponential。

### 技術說法

Poisson、Exponential、Uniform 都常和時間或區間有關，但主詞不同：

```text
Poisson     → 次數
Exponential → 等待時間
Uniform     → 區間內等可能的位置
```

### 流程 / 選擇流程

```text
事件次數 + 固定時間 / 空間 + 平均率 λ → Poisson
下一次事件等待時間 + 無記憶性 → Exponential
連續區間 + 每個位置等可能 → Uniform
```

### 比較表這樣讀

這張表先看「題目問的是」這一欄：

| 題目問的是 | 選 | 直覺 |
| ---------- | --- | ---- |
| 某段時間內發生幾次事件 | Poisson | 數事件次數 |
| 下一次事件要等多久 | Exponential | 等待時間 |
| 區間 `[a,b]` 內任一位置等可能 | Uniform | 平均散在區間內 |

#### Poisson

##### 先懂一句話

Poisson 看「固定範圍內事件出現幾次」。

##### 技術說法

記號：

```text
X ~ Poisson(λ)
```

| 重點     | 說明                                   |
| -------- | -------------------------------------- |
| `λ`      | 平均發生率                             |
| 使用情境 | 固定時間、區域、長度或空間內的事件次數 |
| 常見線索 | 稀有事件、平均每小時幾次、每公里幾個   |
| 平均數   | `λ`                                    |
| 變異數   | `λ`                                    |

##### 一步一步例子

例子：平均每 10 分鐘有 3 通電話，問 10 分鐘內剛好 2 通。

```text
λ = 3
k = 2

P(X = k) = e^-λ λ^k / k!
P(X = 2) = e^-3 × 3^2 / 2!
         = e^-3 × 9 / 2
         = 4.5e^-3
```

`scipy.stats` 讀法：

```python
poisson.pmf(k, mu=λ)  # P(X = k)
poisson.cdf(k, mu=λ)  # P(X ≤ k)
```

#### 指數分佈 (Exponential Distribution) 🔥🔥

##### 先懂一句話

`指數分佈（Exponential Distribution）` 描述「Poisson 事件下一次要等多久」。

##### Everyday Analogy

如果客服電話平均會進來，但你現在關心的不是「10 分鐘內幾通」，而是「下一通電話還要等多久」，就從 Poisson 次數切到 Exponential 等待時間。

##### 技術說法

如果事件以速率 `λ` 發生，則兩次事件之間的等待時間服從指數分佈。

```text
PDF: f(x) = λe^(-λx), x ≥ 0
E(X) = 1/λ
Var(X) = 1/λ²
```

`無記憶性（Memoryless）`：

```text
P(X > s + t | X > s) = P(X > t)
```

白話：已經等了 `s` 分鐘，再等 `t` 分鐘的機率，和一開始就等 `t` 分鐘一樣。

##### 記憶方式

```text
Poisson = 幾次
Exponential = 等多久
```

#### Uniform

##### 先懂一句話

Uniform 看「區間內每個位置等可能」。

##### 技術說法

記號：

```text
X ~ Uniform(a, b)
```

| 重點     | 說明                  |
| -------- | --------------------- |
| 使用情境 | 區間 `[a,b]` 內等可能 |
| PDF      | `1 / (b-a)`           |
| 平均數   | `(a+b) / 2`           |
| 變異數   | `(b-a)^2 / 12`        |

##### 一步一步例子

例子：`X ~ Uniform(2, 6)`，問 `P(3 < X ≤ 5)`。

```text
區間長度 = b - a = 6 - 2 = 4
要算的區間長度 = 5 - 3 = 2
每個位置等可能

P(3 < X ≤ 5) = 目標區間長度 / 總區間長度
             = 2 / 4
             = 0.5
```

`scipy.stats` 讀法：

```python
uniform.pdf(x, loc=a, scale=b-a)
uniform.cdf(x, loc=a, scale=b-a)
```

注意：SciPy 的 `scale` 是區間寬度 `b-a`，不是右界 `b`。例如 `Uniform(2,6)` 要寫 `loc=2, scale=4`。

### 記憶方式

```text
Poisson = 固定區間數幾次，mean = variance = λ
Exponential = 等下一次，mean = 1/λ
Uniform = 區間等可能，scale = b-a
```

### Exam Rule

```text
固定區間內發生幾次 / 平均率 λ → Poisson
Poisson 的 mean = variance = λ
問下一次事件等多久 / 無記憶性 → Exponential
稀有事件 + n ≥ 100 且 np ≤ 10 + λ=np → Poisson 近似 Binomial
區間內每個值等可能 → Uniform
Uniform(a,b) 在 scipy 是 loc=a, scale=b-a
```

### Quick Check

**Q.** 平均每 10 分鐘有 3 通客服電話，問 10 分鐘內剛好 2 通的機率，最像哪個分佈？

答案：卜瓦松分佈。因為題目是在固定時間內問事件發生次數，且給了平均率。

## 7. 分佈關係、CLT 與 scipy.stats 讀碼 🔥🔥

### 先懂一句話

考試常把分佈關係和程式碼混在一起考：你要看得懂 Binomial 何時近似 Poisson 或 Normal、CLT 的主詞是誰，以及 `.pmf`、`.pdf`、`.cdf` 到底在算什麼。

### Everyday Analogy

像看外送平台報表：

- 單筆送達時間可能很偏，有些很快、有些很慢。
- 但如果每天都抽很多筆訂單算「平均送達時間」，很多天的平均值會比較接近鐘形。
- 這就是 `中央極限定理（Central Limit Theorem, CLT）` 在講的「樣本平均數」。

`.pmf`、`.pdf`、`.cdf` 則像報表上的三個按鈕：問離散單點按 `.pmf`，問連續密度按 `.pdf`，問累積到某個值為止按 `.cdf`。

### 先問自己一個問題

題目現在是在問哪一層？

```text
1. 分佈選擇：Binomial / Poisson / Normal / Uniform？
2. 近似：Binomial 能不能用 Poisson 或 Normal 代替？
3. CLT：題目說的是樣本平均數，還是原始資料？
4. 程式碼：pmf / pdf / cdf 代表什麼？
```

### 技術說法

`近似（Approximation）` 是用比較容易處理的分佈去近似原本的分佈。CLT 則是在說：樣本數夠大時，樣本平均數的抽樣分佈會趨近常態。

### 流程 / 選擇流程

```text
已選分佈
→ 判斷是否有近似條件
→ 判斷是原始資料還是樣本平均
→ 讀 scipy.stats 的分佈與函式
```

#### Binomial / Poisson / Normal 的差別

##### 比較表這樣讀

先看題目有沒有固定 `n`、平均率 `λ`、或鐘形連續資料：

```text
Binomial = 固定試驗次數 n，每次成功/失敗，問成功幾次
Poisson  = 固定時間/空間內，事件平均發生率 λ，問發生幾次
Normal   = 連續數值或大樣本近似，形狀像鐘形曲線
```

| 題目關鍵字                            | 選哪個分佈 | 直覺                       |
| ------------------------------------- | ---------- | -------------------------- |
| `n` 次、成功率 `p`、成功次數          | Binomial   | 固定做幾次，看成功幾次     |
| `λ`、rate、每小時/每天/每區域發生幾次 | Poisson    | 固定範圍內，看事件發生次數 |
| `μ`、`σ`、連續測量值、bell-shaped     | Normal     | 連續數值落在哪個範圍       |

#### Binomial 的兩種近似不要混淆

##### 先問自己一個問題

題目是「成功很罕見」還是「成功與失敗都夠多」？

##### 技術說法

```text
Binomial → Poisson
條件：n ≥ 100 且 np ≤ 10（λ = np）
直覺：試驗次數很多，但成功事件很罕見
結論：Binomial(n,p) 可用 Poisson(λ=np) 近似

Binomial → Normal
條件：np ≥ 5 且 n(1-p) ≥ 5
直覺：成功和失敗都夠多，分布形狀會接近鐘形
結論：Binomial 可用 Normal 近似
```

##### 一步一步例子

例子 1：Binomial → Poisson

```text
n = 200, p = 0.02
np = 200 × 0.02 = 4

檢查:
n ≥ 100 → 200 ≥ 100，成立
np ≤ 10 → 4 ≤ 10，成立

所以 Binomial(n=200, p=0.02)
可用 Poisson(λ=np=4) 近似
```

例子 2：Binomial → Normal

```text
n = 100, p = 0.4
np = 100 × 0.4 = 40
n(1-p) = 100 × 0.6 = 60

檢查:
np ≥ 5 → 40 ≥ 5，成立
n(1-p) ≥ 5 → 60 ≥ 5，成立

所以可用 Normal 近似 Binomial
```

> **連續性修正 (Continuity Correction) 🔥🔥 考試陷阱**：二項分佈是離散的，用常態近似時需加 ±0.5 修正。例如 P(X ≤ k) ≈ P(Z ≤ (k + 0.5 − μ) / σ)。忽略此修正會讓計算結果出現誤差，是常見考試陷阱。

#### 中央極限定理（Central Limit Theorem, CLT）

##### 先懂一句話

CLT 不是說原始資料會變常態；它是說「樣本平均數」的分佈會趨近常態。

##### 技術說法

```text
不管原始母體分佈長怎樣，
只要樣本數夠大（一般教科書以 n ≥ 30 作為「樣本夠大」的標準），
樣本平均數的抽樣分佈會趨近常態分佈。
```

更精確地說：

```text
樣本平均數的平均約為 μ
樣本平均數的變異約為 σ² / n
```

##### 一步一步例子

```text
假設外送時間母體平均 μ = 30 分鐘，標準差 σ = 12 分鐘
每次抽 n = 36 筆訂單算平均

樣本平均數的平均 = μ = 30
樣本平均數的變異 = σ² / n
                 = 12² / 36
                 = 144 / 36
                 = 4
樣本平均數的標準差 = sqrt(4) = 2
```

重點：變成接近常態的是「很多次抽樣得到的平均數」，不是每一筆外送時間。

#### scipy.stats 一眼判讀

##### 比較表這樣讀

先看點號後面的函式：`.pmf`、`.pdf`、`.cdf` 比分佈名稱更能告訴你題目在問哪種機率。

| 看到        | 讀成                     |
| ----------- | ------------------------ |
| `.pmf(...)` | 離散型單點機率 `P(X=x)`  |
| `.pdf(...)` | 連續型密度，不是單點機率 |
| `.cdf(...)` | 累積機率 `P(X≤x)`        |
| `norm`      | 常態分佈                 |
| `binom`     | 二項分佈                 |
| `poisson`   | 卜瓦松分佈               |
| `uniform`   | 均勻分佈                 |

##### 一步一步例子

```python
binom.pmf(3, n=10, p=0.2)
```

```text
binom → Binomial
pmf → P(X = x)
第一個參數 3 → k = 3
所以讀成：10 次試驗、成功率 0.2，剛好成功 3 次的機率
```

```python
norm.cdf(85, loc=70, scale=10)
```

```text
norm → Normal
cdf → P(X ≤ x)
x = 85
loc = μ = 70
scale = σ = 10
所以讀成：常態分佈下 X ≤ 85 的累積機率
```

### 記憶方式

```text
pmf = point for discrete
pdf = density for continuous
cdf = cumulative up to x

CLT 主詞 = 樣本平均數
Poisson 近似 = 很多次但成功很少
Normal 近似 = 成功和失敗都夠多
```

### Exam Rule

```text
n ≥ 100 且 np ≤ 10（λ=np）→ Binomial 用 Poisson 近似
np ≥ 5 且 n(1-p) ≥ 5 → Binomial 用 Normal 近似
CLT → 樣本平均數趨近常態，不是原始資料變常態
.pmf → P(X=x)
.cdf → P(X≤x)
.pdf → 密度，不是機率
```

### Quick Check

**Q.** CLT 說的是「原始資料都會變成常態」嗎？

答案：不是。CLT 說的是樣本數夠大時，樣本平均數的抽樣分佈會趨近常態。

## 8. Exam Decision Trees

### 8.1 先選 PMF / PDF / CDF

```text
題目問哪種機率？
│
├─ 問 P(X = x)？
│  ├─ X 是離散型 → PMF
│  └─ X 是連續型 → P(X=x)=0
│
├─ 問 P(X ≤ x)、最多、不超過、以下？
│  └─ CDF
│
└─ 問 P(a < X ≤ b)、介於兩值之間？
   ├─ 可用 CDF 相減 → F(b)-F(a)
   └─ 連續型也可想 PDF 面積
```

### 8.2 先選分佈

```text
題目描述資料型態？
│
├─ 連續型？
│  ├─ 對稱鐘形 / μ / σ² / z-score → Normal
│  └─ 區間 [a,b] 內等可能 → Uniform
│
└─ 離散型？
   ├─ 固定 n 次，每次成功 / 失敗 → Binomial
   └─ 固定時間 / 空間內發生幾次，給 λ → Poisson
```

### 8.3 近似判斷

```text
題目問 Binomial 可否近似？
│
├─ n ≥ 100 且 np ≤ 10、λ=np？
│  └─ 用 Poisson 近似
│
└─ np ≥ 5 且 n(1-p) ≥ 5？
   └─ 用 Normal 近似
```

## 9. Trap Clinic

### Trap 1：PDF 的值就是某點機率

錯。PDF 是密度，不是機率；連續型的單點機率 `P(X=x)` 等於 0。

Exam fix：

```text
連續型 + 單點 → P(X=x)=0
連續型 + 區間 → PDF 面積或 CDF 相減
```

### Trap 2：CDF 是 `P(X=x)`

錯。CDF 是 `F(x)=P(X≤x)`，表示累積到 x 為止。

Exam fix：

```text
最多 / 不超過 / 以下 / ≤ → CDF
剛好等於 + 離散型 → PMF
```

### Trap 3：Normal(μ, σ²) 的 SciPy `scale` 要填 σ²

錯。`norm.pdf` 和 `norm.cdf` 的 `scale` 要填標準差 `σ`，不是變異數 `σ²`。

Exam fix：

```text
數學 Normal(μ, σ²) → scipy norm(..., loc=μ, scale=σ)
```

### Trap 4：二項分佈 PMF 可以省略 `C(n,k)`

錯。`C(n,k)` 表示哪 k 次成功的組合數，漏掉就只算到一種固定順序。

Exam fix：

```text
Binomial PMF → C(n,k)p^k(1-p)^(n-k)
```

### Trap 5：Poisson 只要記得 λ，不用記 mean 和 variance

錯。Poisson 的高頻特徵是 `mean = variance = λ`。

Exam fix：

```text
Poisson(λ) → 平均數 λ，變異數 λ
```

### Trap 6：CLT 說原始資料會變常態

錯。CLT 說的是「樣本平均數的抽樣分佈」趨近常態，不是原始資料本身變常態。

Exam fix：

```text
CLT 主詞 → 樣本平均數
不是 → 單一觀測值 / 原始資料
```

### Trap 7：Uniform 的 scipy `scale` 是右界 b

錯。`uniform(..., loc=a, scale=b-a)`，`scale` 是區間寬度。

Exam fix：

```text
Uniform(a,b) → loc=a, scale=b-a
```

## 10. Practice Questions

### 10.1 隨機變數、PMF / PDF / CDF

**Q1.** 一天收到的客服申訴件數是離散型還是連續型？

答案：離散型。  
理由：件數是可數的整數。

**Q2.** 外送送達時間是離散型還是連續型？

答案：連續型。  
理由：時間可落在連續區間，例如 25.3 分鐘。

**Q3.** `P(X = 4)` 且 X 是離散型，應想到 PMF、PDF 還是 CDF？

答案：PMF。  
理由：PMF 用來讀離散型單點機率。

**Q4.** `F(8)=0.9` 代表什麼？

答案：`P(X≤8)=0.9`。  
理由：CDF 表示累積到某個值為止的機率。

**Q5.** 連續型變數的 `P(X=5)` 通常是多少？

答案：0。  
理由：連續型單點沒有面積，所以單點機率為 0。

### 10.2 四大分佈

**Q6.** 題目描述成績呈左右對稱鐘形，且給平均數與標準差，最像哪個分佈？

答案：常態分佈。  
理由：對稱鐘形、`μ`、`σ` 是 Normal 的典型線索。

**Q7.** 20 通電話中每通成交機率固定，問剛好成交 5 通，最像哪個分佈？

答案：二項分佈。  
理由：固定 `n` 次，每次成功 / 失敗，問成功次數。

**Q8.** 某網站平均每小時當機 0.2 次，問一小時內當機次數，最像哪個分佈？

答案：卜瓦松分佈。  
理由：固定時間內的事件發生次數，且有平均率。

**Q9.** 亂數在 0 到 1 之間每個位置等可能，最像哪個分佈？

答案：均勻分佈。  
理由：連續區間內等可能是 Uniform 的核心線索。

**Q10.** `X ~ Poisson(λ)` 的平均數與變異數是多少？

答案：平均數是 `λ`，變異數也是 `λ`。  
理由：`mean = variance = λ` 是 Poisson 的高頻特徵。

### 10.3 近似、CLT 與 scipy.stats

**Q11.** `n` 很大、`p` 很小、`λ=np` 固定時，Binomial 可用哪個分佈近似？

答案：Poisson。  
理由：這是 Binomial → Poisson 近似的典型條件。

**Q12.** `np ≥ 5` 且 `n(1-p) ≥ 5` 時，Binomial 常可用哪個分佈近似？

答案：Normal。  
理由：這是 Binomial → Normal 近似的常見判斷條件。

**Q13.** CLT 的主詞是原始資料還是樣本平均數？

答案：樣本平均數。  
理由：CLT 說樣本平均數的抽樣分佈會趨近常態。

**Q14.** `binom.pmf(3, n=10, p=0.2)` 讀成什麼？

答案：`P(X=3)`。  
理由：`.pmf` 是離散型單點機率，這裡是剛好成功 3 次。

**Q15.** `poisson.cdf(2, mu=3)` 讀成什麼？

答案：`P(X≤2)`。  
理由：`.cdf` 是小於等於某值的累積機率。

**Q16.** `norm.cdf(85, loc=70, scale=10)` 讀成什麼？

答案：常態分佈下 `P(X≤85)`。  
理由：`norm.cdf` 表示累積到 85 為止的機率。

**Q17.** `uniform.pdf(x, loc=2, scale=4)` 對應的 Uniform 區間是什麼？

答案：`Uniform(2,6)`。  
理由：`loc=a=2`，`scale=b-a=4`，所以 `b=6`。

### 10.4 Mixed Traps

**Q18.** PDF 的數值可以大於 1 嗎？

答案：可以。  
理由：PDF 是密度，不是機率；只要整體面積為 1 即可。

**Q19.** 題目問「最多 4 次成功」，應用 `.pmf(4)` 還是 `.cdf(4)`？

答案：`.cdf(4)`。  
理由：「最多 4 次」是 `P(X≤4)`。

**Q20.** `Normal(50, 25)` 在 SciPy 中 `scale` 是 25 還是 5？

答案：5。  
理由：25 是變異數 `σ²`，`scale` 要填標準差 `σ`。

## Final Oral Recall

考前最後 3 分鐘，把這幾句唸一次：

1. 本課先分離散與連續：次數件數多半離散，時間長度重量多半連續。
2. PMF 是離散單點 `P(X=x)`，PDF 是連續密度，CDF 是累積 `P(X≤x)`。
3. Normal 是連續、對稱鐘形，參數是 `μ, σ²`，z-score 是 `(x-μ)/σ`。
4. Binomial 是固定 `n` 次成功 / 失敗，PMF 不能漏 `C(n,k)`。
5. Poisson 是固定區間內事件次數，參數 `λ` 同時是平均數與變異數。
6. Uniform 是 `[a,b]` 內等可能，SciPy 寫 `loc=a, scale=b-a`。
7. CLT 講樣本平均數趨近常態，不是原始資料自己變常態。
