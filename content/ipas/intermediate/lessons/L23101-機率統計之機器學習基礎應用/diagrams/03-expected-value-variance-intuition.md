# 期望值與變異直覺圖 — Expected Value & Variance Intuition

> 兩個數字，分別告訴你「分佈在哪裡」和「分佈多分散」

## ASCII 直覺圖

```
分佈 A（低變異）         分佈 B（高變異）
        ↓ μ=5                    ↓ μ=5
        ▲                    ▲
       ███                  █   █
      █████                █     █
     ███████              █       █
    █████████            █         █
───────────────      ─────────────────
  2  3  4  5  6      0  2  4  6  8  10

期望值相同（μ=5）       期望值相同（μ=5）
變異小 → 預測穩定       變異大 → 預測風險高
```

## 在 ML 模型中的用途

```mermaid
mindmap
  root((ML 統計量))
    期望值 E[X]
      告訴模型：中心在哪裡
      Loss function 的目標
      均值預測基準線
    變異數 Var[X]
      告訴模型：資料多分散
      特徵縮放依據
      模型不確定性
    標準差 σ
      Var[X] 的平方根
      同單位→可解讀
      Z-score 分母
```

## 公式速查

| 指標 | 定義 | 直覺 |
|---|---|---|
| 期望值 E[X] | Σ xᵢ·P(xᵢ) | 加權平均中心 |
| 變異數 Var[X] | E[(X - μ)²] | 偏離中心的平均平方距離 |
| 標準差 σ | √Var[X] | 與原始資料同單位 |

> 🔑 考試看到「模型預測的中心傾向」→ Expected value；看到「資料的分散程度」→ Variance / Std Dev
