# MLP 前向傳播張量流程

## 網路結構與維度變換

```
輸入層 (Input)         隱藏層 1 (H1)          隱藏層 2 (H2)          輸出層 (Output)
[batch, 784]           [batch, 256]            [batch, 128]           [batch, 10]
     │                       │                       │                      │
784 神經元              256 神經元              128 神經元             10 神經元
(MNIST 像素)          (ReLU 啟動)             (ReLU 啟動)          (Softmax 啟動)
     │                       │                       │                      │
     └──── a = σ(Wx+b) ─────►│──── a = σ(Wx+b) ────►│─── a = σ(Wx+b) ────►│
```

## 每層運算細節

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  Layer 1:  Input → H1                                                       │
│  ─────────────────────────────────────────────────────────────────────────  │
│  W₁ shape:  [784, 256]      b₁ shape:  [256]                               │
│  運算:  a₁ = ReLU(x · W₁ + b₁)                                             │
│  輸入:  [batch, 784]  ──►  輸出:  [batch, 256]                              │
│  參數量:  784 × 256 + 256 = 200,704 + 512 = 201,216                        │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  Layer 2:  H1 → H2                                                          │
│  ─────────────────────────────────────────────────────────────────────────  │
│  W₂ shape:  [256, 128]      b₂ shape:  [128]                               │
│  運算:  a₂ = ReLU(a₁ · W₂ + b₂)                                            │
│  輸入:  [batch, 256]  ──►  輸出:  [batch, 128]                              │
│  參數量:  256 × 128 + 128 = 32,768 + 128 = 32,896                          │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  Layer 3:  H2 → Output                                                      │
│  ─────────────────────────────────────────────────────────────────────────  │
│  W₃ shape:  [128, 10]       b₃ shape:  [10]                                │
│  運算:  ŷ = Softmax(a₂ · W₃ + b₃)                                          │
│  輸入:  [batch, 128]  ──►  輸出:  [batch, 10]                               │
│  參數量:  128 × 10 + 10 = 1,280 + 10 = 1,290                               │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 參數量彙總

| 層   | W 形狀       | b 形狀 | 計算式              | 參數量   |
|------|------------|--------|---------------------|---------|
| L1   | [784, 256] | [256]  | 784×256 + 256       | 201,216 |
| L2   | [256, 128] | [128]  | 256×128 + 128       | 32,896  |
| Out  | [128, 10]  | [10]   | 128×10  + 10        | 1,290   |
| **總計** | —      | —      | —                   | **235,402** |

> 考試快判：每層參數量 = 輸入維度 × 輸出維度 + 輸出維度（偏置項）

## PyTorch 實作對照

```python
import torch
import torch.nn as nn

class MLP(nn.Module):
    def __init__(self):
        super().__init__()
        # nn.Linear(in_features, out_features) — 自動建立 W 與 b
        self.fc1 = nn.Linear(784, 256)   # 參數量 201,216
        self.fc2 = nn.Linear(256, 128)   # 參數量  32,896
        self.fc3 = nn.Linear(128, 10)    # 參數量   1,290
        self.relu = nn.ReLU()

    def forward(self, x):
        # x shape: [batch, 784]
        x = self.relu(self.fc1(x))   # [batch, 784] → [batch, 256]
        x = self.relu(self.fc2(x))   # [batch, 256] → [batch, 128]
        x = self.fc3(x)              # [batch, 128] → [batch,  10]
        return x                     # 輸出 logits；搭配 CrossEntropyLoss

model = MLP()
# 驗證參數量
total = sum(p.numel() for p in model.parameters())
print(f"總參數量: {total:,}")  # 235,402
```

## 前向傳播公式拆解

```
a = σ(Wx + b)

W  — 權重矩陣 (weight matrix)；形狀 [in, out]
x  — 輸入向量/矩陣；形狀 [batch, in]
b  — 偏置向量 (bias)；形狀 [out]；廣播加到每個 batch 樣本
σ  — 啟動函數 (activation function)
   ├── 隱藏層常用 ReLU(z) = max(0, z)，避免梯度消失
   └── 輸出層：多分類用 Softmax，二分類用 Sigmoid，迴歸用 Linear (無啟動)
```

## 🔥🔥🔥 考試重點

- **參數量公式**：`(前層神經元數 × 後層神經元數) + 後層神經元數`（偏置項必加）
- **輸入層不計入參數**：784 個輸入節點本身無 W，參數從第一個 Linear 層開始算
- **nn.Linear 自動含 bias**：預設 `bias=True`；若題目說「無偏置」需另外指定
- **Softmax 不放在 forward() 最後**：搭配 PyTorch `CrossEntropyLoss` 時，loss 函數已內建 Softmax，重複套用反而出錯
