# 圖表 4：四五分之一法則計算實例（Four-Fifths Rule Worked Example）

> 用途：展示從混淆矩陣到 Disparate Impact Ratio 的完整計算步驟，這是最高頻的考試計算題型。

## 情境：銀行信貸審核模型

一家銀行使用機器學習模型審核個人信貸申請。審核結果如下：

### 步驟 1：讀取各群體的混淆矩陣

**群體 A（優勢群體，例：本國籍申請者）— 100 人**

```
實際 / 預測     | 預測核准(+) | 預測拒絕(-)
----------------|------------|------------
實際合格(+)     |   TP = 55  |   FN = 5
實際不合格(-)   |   FP =  5  |   TN = 35
```

**群體 B（弱勢群體，例：外籍申請者）— 100 人**

```
實際 / 預測     | 預測核准(+) | 預測拒絕(-)
----------------|------------|------------
實際合格(+)     |   TP = 35  |   FN = 5
實際不合格(-)   |   FP =  5  |   TN = 55
```

### 步驟 2：計算 Selection Rate（核准率）

```
群體 A 核准率 = (TP + FP) / 總人數 = (55 + 5) / 100 = 60/100 = 0.60 = 60%
群體 B 核准率 = (TP + FP) / 總人數 = (35 + 5) / 100 = 40/100 = 0.40 = 40%
```

### 步驟 3：計算 Disparate Impact Ratio（DI）

```
DI = 弱勢群體核准率 / 優勢群體核准率
DI = 0.40 / 0.60
DI = 0.667
```

### 步驟 4：判斷是否違反四五分之一法則

```
四五分之一法則（Four-Fifths Rule / 80% Rule）：
    DI < 0.8 → 可能存在不合理歧視，需要調查或採取緩解措施
    DI ≥ 0.8 → 通過門檻（但不代表完全公平）

本例：DI = 0.667 < 0.8 → ❌ 違反四五分之一法則
```

### 步驟 5：決策

```
DI = 0.667 < 0.8
↓
需要進一步調查偏誤來源，並考慮採取緩解策略：
    - Pre-processing：對群體 B 樣本進行 Reweighting（提高 sample_weight）
    - Post-processing：對群體 B 在邊界附近的拒絕案例重新審核（Reject-Option）
```

## 延伸：同時計算其他指標

從上面的混淆矩陣也可以計算：

```python
# 群體 A
TP_A, FN_A, FP_A, TN_A = 55, 5, 5, 35
TPR_A = TP_A / (TP_A + FN_A)  # 55/(55+5) = 0.917
FPR_A = FP_A / (FP_A + TN_A)  # 5/(5+35)  = 0.125

# 群體 B
TP_B, FN_B, FP_B, TN_B = 35, 5, 5, 55
TPR_B = TP_B / (TP_B + FN_B)  # 35/(35+5) = 0.875
FPR_B = FP_B / (FP_B + TN_B)  # 5/(5+55)  = 0.083

# Equal Opportunity Difference（EOD）
EOD = TPR_B - TPR_A  # 0.875 - 0.917 = -0.042（差異小，接近公平）

# Demographic Parity Difference（DPD）
sel_A = (TP_A + FP_A) / 100  # 0.60
sel_B = (TP_B + FP_B) / 100  # 0.40
DPD = sel_B - sel_A           # 0.40 - 0.60 = -0.20（差異大）
```

**結論：DI 和 DPD 都顯示有顯著差異，但 EOD 差異較小。這是因為兩組的「真正合格申請人」實際上核准率相近，主要問題在於整體核准率（含假陽性）的差距。**

## 考試速記

```
DI 計算三步驟：
1. Selection Rate = (TP+FP)/N per group
2. DI = 弱勢率 / 優勢率（小的除以大的）
3. DI < 0.8 → 違反四五分之一法則

記憶公式：弱勢分子，優勢分母，低於八成要處理
```

## 常見陷阱

```
❌ 錯誤：DI = 優勢群體率 / 弱勢群體率（分子分母顛倒）
✅ 正解：DI = 弱勢群體率 / 優勢群體率（弱勢在分子）

❌ 錯誤：DI < 0.9 才違規
✅ 正解：DI < 0.8（四五分之一，即 4/5 = 80%）

❌ 錯誤：DI 越高越好
✅ 正解：DI 理想為 1.0（完全相同），< 0.8 才需要警戒
```
