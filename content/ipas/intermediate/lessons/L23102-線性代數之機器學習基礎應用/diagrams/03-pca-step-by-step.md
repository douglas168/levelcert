# PCA 降維步驟流程 — PCA Step-by-Step

## PCA 5 步驟流程圖

```
輸入：原始資料矩陣 X (n × d)
         │
         ▼
┌─────────────────────────────┐
│  步驟 ①  資料中心化           │
│  X_c = X - mean(X, axis=0)  │
│  (每個特徵減去其平均值)        │
└─────────────┬───────────────┘
              │
              ▼
┌─────────────────────────────┐
│  步驟 ②  計算共變異數矩陣     │
│  C = X_cᵀ X_c / (n-1)      │
│  維度：(d × d) 對稱矩陣      │
└─────────────┬───────────────┘
              │
              ▼
┌─────────────────────────────┐
│  步驟 ③  特徵值分解          │
│  C v = λ v                  │
│  → 得到 d 個特徵值 λ₁≥λ₂≥...│
│  → 得到 d 個特徵向量 v₁,v₂,...│
└─────────────┬───────────────┘
              │
              ▼
┌─────────────────────────────┐
│  步驟 ④  選前 k 個主成分     │
│  選使 Σλᵢ/Σλ ≥ 0.95 的最小 k │
│  W = [v₁, v₂, ..., vₖ]     │
│  維度：(d × k) 投影矩陣      │
└─────────────┬───────────────┘
              │
              ▼
┌─────────────────────────────┐
│  步驟 ⑤  投影到 k 維子空間   │
│  X_pca = X_c @ W            │
│  維度：(n × d) → (n × k)    │
└─────────────┬───────────────┘
              │
              ▼
輸出：降維後資料矩陣 X_pca (n × k)
```

## 口訣記憶

```
中  共  分  排  投
↓   ↓   ↓   ↓   ↓
中心化 → 共變異數矩陣 → 分解特徵值 → 排序選k → 投影
```

## 2D 範例追蹤

```
原始資料 (4 樣本, 2 特徵):
X = [[2.5, 2.4],    mean = [2.0, 2.2]
     [0.5, 0.7],
     [2.2, 2.9],
     [1.9, 2.2]]

① 中心化後:
X_c = [[0.5,  0.2],
       [-1.5, -1.5],
       [0.2,  0.7],
       [-0.1, 0.0]]

② 共變異數矩陣 C (2×2):
   C = [[0.62, 0.61],
        [0.61, 0.72]]

③ 特徵值分解:
   λ₁ = 1.28,  v₁ = [0.68, 0.74]   ← 第一主成分方向
   λ₂ = 0.05,  v₂ = [-0.74, 0.68]  ← 第二主成分方向

④ 選 k=1 (λ₁ 解釋 1.28/1.33 ≈ 96% 變異量)

⑤ 投影:
   X_pca = X_c @ [[0.68], [0.74]]   → (4×1) 向量
         = [[-0.83], [2.11], [-0.17], [0.07]]
```

## sklearn API 對應

```python
from sklearn.decomposition import PCA

pca = PCA(n_components=2)
X_pca = pca.fit_transform(X)

# 等價於手動 5 步驟：
# ① X_c = X - X.mean(axis=0)          (fit 內部處理)
# ② C = np.cov(X_c.T)                 (fit 內部處理)
# ③ eigenvalues, eigenvectors = ...   (sklearn 用 SVD 實作，更穩定)
# ④ top-k eigenvectors → W            (n_components 決定 k)
# ⑤ X_pca = X_c @ W                   (transform 做這步)

# 重要屬性:
pca.explained_variance_ratio_          # 每個主成分解釋的變異比例
pca.explained_variance_ratio_.cumsum() # 累積解釋變異量
pca.components_                        # 主成分方向向量 (k × d)
```
