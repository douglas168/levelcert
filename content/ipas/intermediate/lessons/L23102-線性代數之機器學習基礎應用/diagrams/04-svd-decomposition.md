# SVD 三矩陣分解解剖 — SVD Decomposition Anatomy

## SVD 結構圖

```
任意矩陣 A (m × n)  =  U × Σ × Vᵀ

     A         =    U    ×    Σ    ×   Vᵀ
  (m × n)          (m×m)   (m×n)   (n×n)

全矩陣版 (full_matrices=True):
┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐
│         │   │         │   │σ₁       │   │         │
│  A      │ = │    U    │ × │  σ₂     │ × │   Vᵀ    │
│(m × n)  │   │(m × m)  │   │    σ₃   │   │(n × n)  │
│         │   │         │   │      0  │   │         │
└─────────┘   └─────────┘   └─────────┘   └─────────┘

精簡版 (full_matrices=False, k = min(m,n)):
┌─────────┐   ┌──────┐   ┌──────┐   ┌──────┐
│         │   │      │   │σ₁    │   │      │
│  A      │ = │  U   │ × │  σ₂  │ × │  Vᵀ  │
│(m × n)  │   │(m×k) │   │    σ₃│   │(k×n) │
│         │   │      │   │      │   │      │
└─────────┘   └──────┘   └──────┘   └──────┘
              左奇異向量   奇異值      右奇異向量
              (列空間方向) (大→小排列)  (行空間方向)

其中 σ₁ ≥ σ₂ ≥ σ₃ ≥ ... ≥ 0  (奇異值非負，由大到小)
```

## SVD 與 PCA 的關係

```
PCA 流程:        SVD 捷徑:
X → X_c →        X_c = U × Σ × Vᵀ
  共變異數矩陣 C →          ↓
  特徵值分解 →    右奇異向量 Vᵀ 的行 = PCA 主成分方向
  選前k主成分      奇異值 σᵢ = √((n-1) × λᵢ)
                  ↓
    等價!         sklearn PCA 底層用 SVD 計算 (數值更穩定)
```

## 低秩近似（Low-rank Approximation）

```
原始矩陣 A (m × n) — 完整資訊

保留前 k 個奇異值的近似：
A_k = U_k × Σ_k × Vᵀ_k

        全部奇異值:   σ₁    σ₂    σ₃    σ₄ ...
                     大    中    小    很小

        保留 k=2:    [σ₁    σ₂   | 0     0   ...]
                     └──────────┘
                        保留部分

儲存空間比較：
  原始: m × n 個數字
  近似: m×k + k + k×n = k(m+n+1) 個數字

當 k << min(m,n) 時，大幅節省空間！

圖像壓縮範例 (500×500 像素圖像):
  k=10:  10×(500+500+1) = 10,010 個數字 (原 250,000 的 4%)
  重建品質: 通常 ~80% 視覺還原度
```

## 考試快判

```
問：SVD 分解輸出三個矩陣，從左到右是？
答：U (左奇異向量) → Σ (奇異值對角矩陣) → Vᵀ (右奇異向量轉置)

問：SVD 和 PCA 最大的差別？
答：SVD 適用任意 m×n 矩陣；PCA 只適用方陣（共變異數矩陣）
   （但 sklearn PCA 底層其實用 SVD 實作）

問：低秩近似保留什麼？
答：前 k 個最大奇異值（及對應的奇異向量）

程式碼：
U, S, Vt = np.linalg.svd(A, full_matrices=False)
# S 是向量（不是矩陣），要用 np.diag(S) 才能做矩陣乘法
A_k = U[:, :k] @ np.diag(S[:k]) @ Vt[:k, :]
```
