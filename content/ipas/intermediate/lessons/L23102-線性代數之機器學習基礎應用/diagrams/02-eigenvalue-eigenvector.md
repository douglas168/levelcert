# 特徵值與特徵向量幾何圖解 — Eigenvalue & Eigenvector Visualization

## 核心定義：Av = λv

```
矩陣 A 作用在向量 v 上的兩種情況：

情況一：一般向量 (方向改變)
  v = [1, 0]               Av = [2, 1]
       ↑                        ↗ (方向改變了!)
    原始向量              變換後向量

情況二：特徵向量 (方向不變，只拉伸)
  v = [1, 1]               Av = [3, 3] = 3 × [1, 1]
       ↗                        ↗↗ (方向不變，拉伸了 3 倍!)
  特徵向量                 λ=3 的拉伸

幾何圖解：
  x₂ ↑
  3  │         ★ Av = 3v (特徵向量 v 被拉伸)
  2  │
  1  │  ● v (特徵向量方向)
  0  └──────────→ x₁
     0  1  2  3

  → v 指向的「方向」在 A 的作用下保持不變
  → λ = 3 是「拉伸倍數」
```

## 共變異數矩陣的特徵向量 = 資料最大變異方向

```
原始資料散點（2D）：
  x₂↑   ·  ·
     · ·  · ·
    · ·   ·       ← 資料沿左下→右上方向分散
     · ·
  0 └──────────→ x₁

第一特徵向量 (PC1)：資料最大變異方向
  x₂↑   ·  ·
     · ·  · ·
    · ·  ↗ ·    ← PC1 沿資料最長軸方向
     · ·
  0 └──────────→ x₁

第二特徵向量 (PC2)：與 PC1 正交，第二大變異方向
  x₂↑   ·↑ ·
     · ·↑ · ·
    · ·  ↗ ·    ← PC2 ⊥ PC1
     · ↑
  0 └──────────→ x₁

特徵值 λ₁ > λ₂ > 0
  λ₁ = PC1 方向的變異量（大）
  λ₂ = PC2 方向的變異量（小）

解釋變異比例 = λᵢ / (λ₁ + λ₂ + ... + λₙ)
```

## 考試快判

```
Q: "矩陣 A 的特徵值是什麼？"
A: 純量 (scalar) — 一個數字，代表拉伸倍數

Q: "矩陣 A 的特徵向量是什麼？"
A: 向量 (vector) — 一個方向，在 A 作用下方向不變

Q: "共變異數矩陣的最大特徵值對應的特徵向量代表什麼？"
A: 資料變異量最大的方向（第一主成分方向）

⚠️  陷阱：特徵值 ≠ 特徵向量，前者是純量，後者是向量
```
