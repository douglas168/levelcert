# 向量與矩陣幾何直覺 — Vector & Matrix Geometry

## 圖一：樣本 = 向量，資料集 = 矩陣

```
特徵空間 (2D 示意)
  x₂ ↑
  4 │         ★ 樣本C (1, 4)
  3 │   ● 樣本B (2, 3)
  2 │ ◆ 樣本A (1, 2)
  1 │
  0 └─────────────→ x₁
    0   1   2   3

資料矩陣 X (3 樣本 × 2 特徵):
       x₁  x₂
樣本A: [ 1   2 ]   ← 向量 [1, 2]
樣本B: [ 2   3 ]   ← 向量 [2, 3]
樣本C: [ 1   4 ]   ← 向量 [1, 4]
```

## 圖二：點積 = 相似度量

```
向量 a = [3, 4]   |a| = 5
向量 b = [4, 3]   |b| = 5

點積 (Dot Product):
  a · b = 3×4 + 4×3 = 24

餘弦相似度 (Cosine Similarity):
  cos(θ) = (a · b) / (|a| × |b|)
          = 24 / (5 × 5)
          = 0.96   ← 方向幾乎一致

θ ≈ 16° (夾角小 → 相似度高)

       x₂↑
     4 │ /a=[3,4]
     3 │/  /b=[4,3]
     2 │  /
     1 │ /θ≈16°
     0 └──────→ x₁
       0  1  2  3  4

⚠️  cos(θ) ∈ [-1, 1]，並非 [0, 1]
    負值 → 方向相反（反義詞在 NLP embedding 空間中）
```

## 圖三：矩陣乘法 = 線性變換（維度追蹤）

```
輸入矩陣 X   ×   投影矩陣 W   =   輸出矩陣 X'
  (n × d)       (d × k)           (n × k)

範例：
  (100 × 10)  ×  (10 × 2)  =  (100 × 2)
  ↑                             ↑
  100筆樣本                     100筆樣本
  10個特徵                      2個特徵 (降維!)

維度規則：(n × d) × (d × k) → (n × k)
          內側相消 (d)，外側保留 (n, k)

考試快判：矩陣乘法後輸出維度 = 左矩陣列數 × 右矩陣欄數
```
