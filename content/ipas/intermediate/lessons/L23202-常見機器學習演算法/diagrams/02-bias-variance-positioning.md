# 圖表 2：偏差-方差定位圖

> **用途：** 六大演算法在高/低偏差 × 高/低方差四象限的定位，幫助判斷過擬合/欠擬合診斷與調參方向。

---

## 偏差（Bias）vs 方差（Variance）四象限

```
高方差（Variance）
     ↑
     │   高偏差 + 高方差        低偏差 + 高方差
     │   ┌─────────────────┬─────────────────┐
     │   │                 │  • k-NN (k=1)   │
     │   │    （不常見）    │  • Decision Tree │
     │   │                 │    （無限制深度）  │
     │   │                 │                 │
     ├───┤─────────────────┼─────────────────┤
     │   │   高偏差 + 低方差│  低偏差 + 低方差 │
     │   │  • k-NN (k大)   │                 │
     │   │  • Linear Reg   │  • Random Forest │
     │   │    (欠擬合)      │  • SVM (RBF)    │
     │   │                 │  • Logistic Reg  │
     │   │                 │    (調好)        │
     │   └─────────────────┴─────────────────┘
     │
低方差─────────────────────────────────────────→ 低偏差（Bias）
```

---

## 各演算法偏差-方差特性說明

| 演算法 | 預設偏差 | 預設方差 | 說明 |
|---|---|---|---|
| Linear Regression | 高 | 低 | 模型過簡（線性假設），對複雜關係欠擬合 |
| Ridge / Lasso | 高→中 | 低 | 正則化進一步增偏差、降方差 |
| Logistic Regression | 中 | 低 | 線性邊界，但分類場景通常夠用 |
| Decision Tree（深） | 低 | 高 | 過擬合訓練資料，方差大 |
| Decision Tree（淺） | 高 | 低 | max_depth 小 → 欠擬合 |
| Random Forest | 中 | 低 | Bagging 大幅降低方差，略增偏差 |
| SVM（RBF） | 低 | 中 | Kernel 捕捉非線性，C 調好可平衡 |
| k-NN（k=1） | 低 | 高 | 完全記住訓練資料 |
| k-NN（k 大） | 高 | 低 | 邊界太平滑，欠擬合 |

---

## 調參對偏差-方差的影響

```
調整參數         偏差方向      方差方向
────────────────────────────────────────
增大 alpha（L1/L2）  ↑偏差      ↓方差
增大 C（SVM/LR）    ↓偏差      ↑方差
減小 k（k-NN）      ↓偏差      ↑方差
增大 max_depth（DT）↓偏差      ↑方差
增大 n_estimators（RF）不變    ↓方差（趨穩定）
```

---

## 考試快判

- **訓練 acc 高 + 測試 acc 低** → 低偏差 + 高方差 → 過擬合 → 增加正則化 / 減少複雜度
- **訓練 acc 低 + 測試 acc 低** → 高偏差 + 低方差 → 欠擬合 → 增加複雜度 / 減少正則化
- **Random Forest 相對 Decision Tree** → 偏差略高但方差大幅降低 → 泛化更好
