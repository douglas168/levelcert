# 圖表 5：SVM 間隔幾何示意

> **用途：** Hard margin / Soft margin / Kernel trick 三層幾何圖，標示支持向量位置與超平面關係，幫助理解 SVM 最大間隔直覺與 C 參數意義。

---

## 1. Hard Margin SVM（線性可分情境）

```
特徵空間

  Class A (●)                   Class B (○)

  ●                                        ○
     ●                                 ○
        ●    ←─── margin ───→       ○
           ● ┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄ ○      ← 支持向量（Support Vectors）
              ══════════════════      ← 決策邊界（Decision Boundary）
           ● ┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄ ○      ← 支持向量
        ●                        ○
     ●                               ○
  ●                                      ○

  ← margin = 2/‖w‖，越大越好 →
```

**關鍵概念：**
- 決策邊界由最靠近的點（支持向量）決定
- 優化目標：最大化 2/‖w‖（等價於最小化 ‖w‖²）
- Hard Margin 完全不容許任何點越界 → 只適用線性完全可分資料

---

## 2. Soft Margin SVM（真實資料 — 允許誤差）

```
特徵空間（有噪音和重疊）

  ●                                        ○
     ●                                 ○
        ●         margin          ○
           ● ┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄ ○
              ══════════════════   ← 決策邊界
           ● ┄┄┄┄┄┄┄┄┄┄ ○ ┄
        ●                  ↑          ○
     ●                  誤分類容忍      ○
  ●                    (ξ > 0，鬆弛變數)

```

**C 參數的直觀效果：**

```
C 很大（如 100）：            C 很小（如 0.01）：
「不容忍任何誤分類」          「允許較多誤分類」

  ●  ●                         ●  ●
     ● ══ ○                      ══════○
        ○                    ●    ○
  → 邊界緊，複雜            → 邊界鬆，平滑
  → 過擬合風險高            → 欠擬合風險高
```

| C 值 | 間隔寬度 | 誤分類容忍 | 模型複雜度 | 風險 |
|---|---|---|---|---|
| 大（如 100） | 窄 | 少 | 高 | 過擬合 |
| 小（如 0.01） | 寬 | 多 | 低 | 欠擬合 |
| 中（如 1.0） | 平衡 | 中等 | 中等 | 建議起點 |

## 3. Kernel Trick（非線性可分 → 映射到高維）

```
原始 2D 特徵空間（線性不可分）

  ●●●○○○●●●
    ○○○●●●○○   ← 圓形邊界，線性無法分

         │
         │ Kernel 映射
         │ φ(x) → 高維空間
         ▼

  高維空間（線性可分）

  ●●●                     ○○○●●●○○○
  ─────────────────────────────────
  （在高維空間中，超平面可以完美分開）
```

**三種 Kernel 的幾何直覺：**

```
linear kernel（線性）：       rbf kernel（RBF/Gaussian）：
原始空間直接畫超平面          局部相似度：近的點相似度高
適合：高維文字，已線性可分     適合：任何非線性，最通用
                              gamma 控制「影響範圍」
                              
                              gamma 大 → 每個點只影響近鄰
                                        → 決策邊界複雜 → 過擬合
                              gamma 小 → 影響範圍廣
                                        → 邊界平滑

poly kernel（多項式）：
高維映射到多項式空間
degree=2：二次曲面邊界
degree=3：三次曲面
適合：影像、信號場景
```

---

## Kernel 選擇決策流程

```
資料類型是什麼？
│
├── 高維稀疏（文字、TF-IDF）→ kernel='linear'
│
├── 小/中維度，非線性邊界 → kernel='rbf'（預設起點）
│   ├── 調 C：控制誤分類容忍
│   └── 調 gamma：控制局部影響範圍
│       ├── gamma='scale'（sklearn 預設）= 1/(n_features × X.var())
│       └── gamma='auto' = 1/n_features
│
└── 影像或多項式邊界 → kernel='poly'
    └── degree=3（典型）
```

---

## 考試快判

```
「最大間隔」「支持向量決定邊界」→ SVM 核心原理
「C 越大」→ 邊界越硬 → 過擬合風險高
「C 越小」→ 邊界越鬆 → 欠擬合風險高
「文字分類 + 高維」→ kernel='linear'
「非線性」「一般情況」→ kernel='rbf'
「gamma='scale'」→ sklearn SVC 預設（非 'auto'）
「Kernel Trick」→ 隱式映射到高維，不需要計算高維座標
```
