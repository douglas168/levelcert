# 圖表 4：隨機森林 Bagging vs Boosting 對比

> **用途：** Bagging（Random Forest）與 Boosting（AdaBoost/GBM）流程圖並排比較，幫助學生區分兩類集成方法的訓練方式與誤差修正策略。

---

## Bagging 流程（Random Forest）

```
原始訓練資料 (n 筆)
         │
         ▼
┌────────────────────────────────────────────┐
│         Bootstrap 抽樣（有放回）             │
│  ┌──────────────────────────────────────┐  │
│  │ 抽樣1 → 訓練資料1 → 決策樹1 ────────┐│  │
│  │ 抽樣2 → 訓練資料2 → 決策樹2 ────────┤│  │
│  │ 抽樣3 → 訓練資料3 → 決策樹3 ────────┤│  │
│  │  ...        ...         ...         ││  │
│  │ 抽樣N → 訓練資料N → 決策樹N ────────┘│  │
│  └──────────────────────────────────────┘  │
│                                            │
│   ✅ 各棵樹獨立訓練（可並行）               │
│   ✅ 每棵樹隨機選特徵子集（max_features）   │
└────────────────────────────────────────────┘
         │
         ▼
   投票（分類）/ 平均（迴歸）
         │
         ▼
      最終預測
```

**核心思想：** 靠「多樣性」降低方差（Variance）— 每棵樹的誤差不同，集成後互相抵消。

---

## Boosting 流程（AdaBoost / GBM）

```
原始訓練資料 (n 筆)
         │
         ▼
  弱分類器 1（均等樣本權重）
         │
         │  ← 評估哪些樣本預測錯了
         ▼
  調整樣本權重（錯誤樣本 → 權重增大）
         │
         ▼
  弱分類器 2（重視高權重錯誤樣本）
         │
         │  ← 再次評估錯誤
         ▼
  再調整樣本權重
         │
         ▼
  弱分類器 3 → ... → 弱分類器 N
         │
         │  ❌ 各分類器有依賴（不可並行）
         ▼
  加權投票（每個分類器權重不同）
         │
         ▼
      最終預測
```

**核心思想：** 靠「序列修正」降低偏差（Bias）— 每個弱分類器專注修正前一個的錯誤。

---

## Bagging vs Boosting 並排比較

| 維度 | Bagging（Random Forest） | Boosting（AdaBoost/GBM） |
|---|---|---|
| 訓練方式 | 並行（各棵樹獨立） | 序列（後一棵依賴前一棵） |
| 訓練速度 | 快（可並行） | 慢（必須序列） |
| 主要降低 | **方差（Variance）** | **偏差（Bias）** |
| 過擬合傾向 | 低 | 較高（需控制 learning rate） |
| 主要代表 | Random Forest | AdaBoost、GBM、XGBoost |
| 關鍵超參數 | n_estimators, max_features | n_estimators, learning_rate |
| 對噪音敏感 | 低（Bootstrap 平均掉噪音） | 高（Boosting 會強化噪音樣本） |

---

## AdaBoost 樣本權重調整示意

```
初始：所有樣本權重 = 1/n
         ┌──┬──┬──┬──┬──┐
權重     │1 │1 │1 │1 │1 │  ← 均等
         └──┴──┴──┴──┴──┘

分類器1 預測，樣本3、5 錯：
         ┌──┬──┬──┬──┬──┐
權重     │↓ │↓ │↑↑│↓ │↑↑│  ← 錯誤樣本加重
         └──┴──┴──┴──┴──┘

分類器2 更「關注」權重高的樣本（3、5）
→ 修正分類器1 的錯誤區域
```

---

## 考試快判

```
「並行訓練」「Bootstrap 抽樣」「降低方差」→ Bagging / Random Forest
「序列訓練」「修正前一棵錯誤」「降低偏差」→ Boosting / AdaBoost / GBM
「對噪音更敏感」→ Boosting（強化錯誤樣本）
「學習率（learning rate）」超參數出現 → Boosting（GBM/XGBoost）
```
